\section{Noether 定理的数学提炼}

第 7 章给出了 Noether 定理的完整证明。此处提炼其数学本质，作为四大守恒律推导的共同基础：

\textbf{Noether 第一定理（力学形式）}：$\fbox{[S]}$ 若作用量 $S = \int L(q,\dot{q})\,dt$ 在连续变换 $q_i \to q_i + \varepsilon Q_i(q,\dot{q})$ 下不变（$\delta S = 0$ 对任何时间区间），则：
\begin{equation}
\boxed{\frac{d}{dt}\!\left(\sum_i \frac{\partial L}{\partial\dot{q}_i}Q_i - F\right) = 0}
\end{equation}
其中 $F$ 满足 $L' = L + \varepsilon\,dF/dt$。括号中的量 $= dQ_N/dt = 0$——即 Noether 荷守恒。

\textbf{场论形式}：$\fbox{[S]}$ $\mathcal{L}(\phi,\partial_\mu\phi)$ 在 $\phi \to \phi + \varepsilon\Delta\phi$ 下不变 $\Rightarrow$：
\begin{equation}
\partial_\mu J^\mu = 0, \quad J^\mu \equiv \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi)}\Delta\phi - \mathcal{F}^\mu
\end{equation}

\keypoint{Noether 定理的惊人之处在于它不需要知道运动方程的具体形式——它只需要对称性。\textbf{不需要解运动方程就能知道什么量守恒}。这是物理学中最接近"免费午餐"的东西。}

\section{科普层：守恒律——宇宙的"记账系统"}

守恒律是物理学最接近"会计"的部分。它们不告诉你\textit{什么}会发生——它们告诉你\textit{什么不能}发生。能量不能无中生有。电荷不能凭空消失。就像银行不会允许钱凭空出现或消失——宇宙也有自己严格的"账本"。

但为什么宇宙有这些账本？Noether 在 1918 年给出了最优雅的答案：\textbf{每一个守恒律都是某种深层的、不可见的"不变性"的影子。}

\subsection{滑冰者和我们的直觉}

为什么花滑运动员抱紧手臂时转得更快？这不是你学到的第一条"物理定律"——而是你通过观看体育比赛直觉感知到的。答案就在角动量守恒：转动惯量减小（手臂靠近身体）$\to$ 角速度增大（转得更快）$\to$ 角动量不变。运动员利用的是旋转对称性——当外扭矩为零时，角动量不能变。

\subsection{台球桌上的动量}

白球撞黑球。碰撞前白球以速度 $\mathbf{v}$ 运动，黑球静止。碰撞后白球停下，黑球以速度 $\mathbf{v}$ 运动。总动量从白球完全转移到了黑球——但总量不变。这不是因为台球"有弹性"——即使在完全非弹性碰撞中（球粘在一起），总动量仍然守恒。动量守恒来自空间平移不变性——这里和那里的物理定律一样。如果动量不守恒，宇宙将没有"惯性"的概念。

\subsection{膨胀宇宙的能量问题——Noether 定理的失败案例}

这是 Noether 定理最精彩的"反面应用"。我们的宇宙在膨胀——度规 $g_{\mu\nu}$ 显含时间。因此，\textbf{不存在全局时间平移对称性}。Noether 定理说：没有对称性 $\to$ 没有守恒律。确实，在广义相对论中，\textbf{宇宙的总能量没有全局守恒的定义}。这听起来像物理学的丑闻——但恰恰相反：它完美地证明了 Noether 定理的正确性。对称性不存在 = 守恒律不存在。不多不少。

但为什么我们在太阳系中仍然观测到能量守恒？因为太阳系的尺度远小于宇宙学尺度。在太阳系内，度规的时间变化可以忽略——时间平移对称性在局域近似成立——So能量在局域近似下守恒。这是物理学中一个更大的主题：\textbf{定律本身可以是 emergent——在适当的尺度上涌现}。

\end{lawbox}

\begin{derivationbox}[推导：时间平移对称性到能量守恒]
  \textbf{Step 1} — 时间平移：$t \to t' = t + \varepsilon$，$q_i(t) \to q_i'(t) = q_i(t-\varepsilon) \approx q_i - \varepsilon\dot{q}_i$
  $\implies$ 生成元 $Q_i = -\dot{q}_i$
  \hfill\texttt{[N]} R1 的时间坐标支持平移

  \textbf{Step 2} — 若 $\partial L/\partial t = 0$（不显含时），则变换为对称性：
  $L' - L = \varepsilon\,dL/dt \implies F = -L$

  \textbf{Step 3} — Noether 荷：
  $Q = \sum_i\frac{\partial L}{\partial\dot{q}_i}(-\dot{q}_i) - (-L) = -\sum_i p_i\dot{q}_i + L = -H$

  \textbf{Step 4} — $d(-H)/dt = 0 \implies dH/dt = 0$。定义能量 $E \equiv H = \sum_i p_i\dot{q}_i - L$
  \hfill\texttt{[N]} $\partial L/\partial t = 0$——若 $L$ 显含时，$dH/dt = -\partial L/\partial t \neq 0$

  \textbf{Step 5} — $dE/dt = 0$。\quad\texttt{[S]} $\partial L/\partial t = 0$ + Noether $\Rightarrow$ 能量守恒。
\end{derivationbox}

\keypoint{能量守恒不是"我们观察到能量总是守恒"——它是时间平移对称性的逻辑推论。如果明天物理定律和今天一样（$\partial L/\partial t=0$），则今天和明天的总能量必须相等。反过来，在膨胀宇宙中（度规显含时间），Noether 定理不保证全局能量守恒——而 GR 确实不保证。对称性 = 守恒，不多不少。}

\subsection{物理直觉：为什么动量守恒？}

空间平移不变性说：物理定律在这里和在那里相同。这意味着如果你把整个实验装置平移 1 米，结果不变。Noether 定理的推论：存在一个不随时间改变的量——动量。这不是巧合——这是必然。

但有一个重要的细微之处：\textbf{电磁场本身携带动量}。考虑两个运动的点电荷——它们之间的电磁力并不满足 Newton 第三定律（力的强形式，即力沿连线）。系统的"机械动量"不守恒。但总动量（机械 + 电磁场）守恒。场不是虚拟的——场是真实的物理实体，携带动量、能量和角动量。

\textbf{电磁场动量密度}：$\mathbf{g} = \varepsilon_0\mathbf{E}\times\mathbf{B}$（Poynting 矢量除以 $c^2$）。这解释了为什么光有"辐射压力"——光子将动量转移给被照亮的表面。

\begin{lawbox}[动量守恒]
  $d\mathbf{P}/dt = 0$（孤立系统）
  \quad\textbf{前提}：Noether 定理（空间平移对称性）
\end{lawbox}

\noindent\textbf{变量}：总动量 $\mathbf{P} = \sum_i m_i\mathbf{v}_i$ [M$\cdot$L$\cdot$T$^{-1}$]。

\begin{derivationbox}[推导：空间平移对称性到动量守恒]
  \textbf{Step 1} — 空间平移：$\mathbf{r} \to \mathbf{r} + \varepsilon\hat{\mathbf{n}}$（$\hat{\mathbf{n}}$ 为任意方向）
  $\implies$ 生成元 $Q_i = \hat{\mathbf{n}}$

  \textbf{Step 2} — 若 $V$ 仅依赖粒子间相对距离 $|\mathbf{r}_i - \mathbf{r}_j|$，则 $L$ 平移不变
  \hfill\texttt{[N]} 无外力场破坏空间均匀性

  \textbf{Step 3} — Noether 荷：
  $Q = \sum_i \frac{\partial L}{\partial\dot{\mathbf{r}}_i}\cdot\hat{\mathbf{n}} = (\sum_i m_i\mathbf{v}_i)\cdot\hat{\mathbf{n}}$

  \textbf{Step 4} — 对所有方向成立 $\implies \mathbf{P}$ 矢量守恒
  \hfill\texttt{[N]} 孤立系统（无外力）
\end{derivationbox}

\subsection{角动量的量子化}

在量子力学中（R4），角动量不是连续变量——它是量子化的，单位是 $\hbar$：
\[
\hat{L}^2| \ell, m\rangle = \hbar^2\ell(\ell+1)| \ell, m\rangle, \quad \hat{L}_z| \ell, m\rangle = \hbar m| \ell, m\rangle
\]
其中 $\ell = 0, 1, 2, \ldots$（轨道）或半整数（自旋），$m = -\ell, -\ell+1, \ldots, \ell$。\textbf{轨道的旋转对称性（SO(3)）将角动量量子化——这不是额外公理，而是 R4c（正则对易关系）加 SO(3) 群表示的必然推论。}

更深刻的是：\textbf{自旋角动量}——粒子的"内禀"角动量——不是来自空间旋转，而是来自 Lorentz 群的旋量表示。自旋-1/2 粒子（电子、质子、中子）的波函数在 $2\pi$ 旋转下变号（$\psi \to -\psi$）——这是"旋量"的数学特性，没有经典对应物。但自旋仍然贡献角动量，且总角动量（轨道 + 自旋）在旋转对称下守恒。

\begin{oombox}[角动量守恒的数量级]

    \item \textbf{地球-月球系统}：地球自转的角动量因潮汐摩擦逐渐转移给月球——月球每年远离地球 $\sim 3.8$ cm。总角动量守恒——地球自转减慢的角动量恰好等于月球轨道角动量的增加
    \item \textbf{脉冲星}：蟹状星云脉冲星（$M=1.4 M_\odot$, $R=10$ km, $P=33$ ms）——角动量 $L \sim I\omega \sim 10^{40}$ kg·m$^2$/s。即使经过千年，自转减慢极小——因为空间旋转对称性保护角动量
    \item \textbf{粒子物理}：$\pi^0 \to 2\gamma$ 衰变——光子对的总角动量 = 0（来自 $\pi^0$ 的自旋 0）——这约束了光子的偏振关联。违反角动量守恒的信号会在偏振关联中立即暴露
  \end{itemize}

\subsection{如果电荷不守恒——电子会不会衰变？}

电荷守恒是所有守恒律中检验最严格的之一。如果电荷不守恒，电子可以衰变——$e^- \to \nu_e + \gamma$。Borexino 实验搜索这一衰变——发现电子寿命下限为 $> 6.6 \times 10^{28}$ 年。这比宇宙年龄（$1.38 \times 10^{10}$ 年）长 $10^{18}$ 倍。电子的稳定性是 $U(1)$ 对称性保护的结果——而 $U(1)$ 对称性又由 Noether 定理保护。Noether 定理的实验验证可以精确到宇宙年龄的十亿亿倍。

\section{超越 Noether：拓扑守恒律}

并非所有守恒量都源自 Noether 定理。某些守恒量来自拓扑——场的整体结构，而非局部对称性：

\textbf{磁单极子的 Dirac 量子化}：若磁单极子存在，电荷必须量子化：$eg = n\hbar/2$（$g$ 为磁荷）。这不是 Noether 推论——它来自波函数单值性的拓扑要求。

\textbf{孤子守恒}：Sine-Gordon 方程的解中存在不随时间改变的拓扑孤子——其守恒来自场的边界条件（缠绕数），而非 Lagrangian 的对称性。

\textbf{结论}：Noether 定理捕获了守恒律的"主要来源"——但拓扑和量子反常可以产生（或破坏！）额外的守恒结构。

\section{自发对称性破缺与 Goldstone 定理}

Noether 定理告诉我们对称性 $\to$ 守恒量。但对称性可以被\textbf{自发破缺}——Lagrangian 有对称性，但基态（真空）破坏了这个对称性。这导致两种后果：

\subsection{全局对称性自发破缺：Goldstone 定理}

\textbf{定理（Goldstone 1961）}：$\fbox{[S]}$ 若连续全局对称性自发破缺，则存在零质量、零自旋的玻色子——Goldstone 玻色子。每个破缺的连续对称性生成元$\to$一个无质量模式。

\textbf{实例}：
\begin{itemize}
  \item \textbf{手征对称性自发破缺}：QCD 中 $SU(2)_L\times SU(2)_R \to SU(2)_V$——产生三个 $\pi$ 介子（$\pi^\pm, \pi^0$）作为赝 Goldstone 玻色子。$\pi$ 介子的质量（$\sim 140$ MeV）远小于核子质量（$\sim 940$ MeV）——因为手征对称性是近似的（夸克质量不为零）。
  \item \textbf{铁磁体}：自旋旋转对称性在 Curie 温度以下自发破缺——产生自旋波（magnon）作为 Goldstone 模式。
\end{itemize}

\subsection{局域对称性自发破缺：Higgs 机制}

\textbf{关键区别}：$\fbox{[S]}$ 局域规范对称性自发破缺\textbf{不}产生无质量 Goldstone 玻色子。破缺的规范对称性的"Goldstone 模式"被规范场"吃"掉了——规范场获得质量。这是第 14 章 Higgs 机制的核心，此处仅点明其与 Noether 定理的关系：

\keypoint{Noether 定理 + 自发对称性破缺 = 物理学最深刻的组合之一。Noether 告诉你对称性$\to$守恒律；Goldstone 告诉你破缺的对称性$\to$无质量粒子；Higgs 告诉你局域对称性破缺$\to$有质量的规范玻色子。三者合在一起，构成了从对称性到粒子谱系的完整链条。}

\begin{insightbox}[为什么 Goldstone 定理不矛盾于 Noether 定理？]
  Noether 定理说：对称性 $\to$ 守恒流 $\to$ 守恒荷。Goldstone 定理说：破缺的连续对称性 $\to$ 无质量玻色子。两者不矛盾——Noether 定理仍然保证存在守恒荷（即使对称性自发破缺，Lagrangian 的对称性不变），Goldstone 定理进一步告诉你这个破缺的动力学后果（基态上有无质量激发）。
\end{insightbox}

守恒律不仅仅是"某些量不变"——它们还约束物理过程\textbf{能不能发生}。这是物理学最强大的工具之一：

\section{反常：当对称性在量子层面破缺}

一个经典对称性可能在量子层面被破坏——这称为\textbf{反常}（anomaly）。最著名的例子是轴矢流反常：经典层面 $\partial_\mu J_5^\mu = 0$（手征对称性），但量子修正（三角图）产生 $\partial_\mu J_5^\mu = (e^2/16\pi^2)F_{\mu\nu}\tilde{F}^{\mu\nu} \neq 0$。这导致 $\pi^0 \to 2\gamma$ 的衰变率比经典预言快——实验精确验证了这一反常修正。

\textbf{反常消除}是标准模型自洽性的核心约束：标准模型的夸克-轻子代数恰好使所有规范反常消失（$\sum_i Y_i = 0$, $\sum_i Y_i^3 = 0$ 等）。这不是"选出来的"——自然界如果选择了有反常的粒子内容，量子场论将是不自洽的。\textbf{量子理论的自洽性本身约束了可能的粒子内容——这是"理论选择物质"的最强例证。}

\section*{习题}

\begin{enumerate}
  \item \textbf{[简单]} 从 Noether 定理推导时间平移 $\to$ 能量守恒。哪步要求 $\partial L/\partial t=0$？若 $L$ 显含时，$dH/dt$ 等于什么？
  \item \textbf{[简单]} 两粒子系统 $L=\frac{1}{2}m_1\dot{\mathbf{r}}_1^2+\frac{1}{2}m_2\dot{\mathbf{r}}_2^2 - V(|\mathbf{r}_1-\mathbf{r}_2|)$：空间平移 $\mathbf{r}_i\to\mathbf{r}_i+\varepsilon\hat{\mathbf{n}}$ 是对称性，用 Noether 求守恒量。
  \item \textbf{[中等]} 从场论 Noether 流 $J^\mu = \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi)}\delta\phi$ 出发，对复标量场和 $U(1)$ 变换，导出电荷守恒流并验证 $\partial_\mu J^\mu=0$。
  \item \textbf{[中等]} 若 $V=kx^2+2ky^2$（破坏旋转对称），角动量的哪个分量不守恒？用 Noether 验证。
  \item \textbf{[较难]} 逆 Noether 定理：对一维系统 $H=p^2/(2m)+V(x)$，若 $H$ 守恒，能否反推 $\partial L/\partial t=0$？
  \item \textbf{[挑战]} GR 中无全局时间平移对称（膨胀宇宙），Noether 不保证全局能量守恒。但为什么太阳系仍见能量守恒？GR 中 $\nabla_\mu T^{\mu\nu}=0$ 如何退化到 $\partial_\mu T^{\mu\nu}=0$？
\end{enumerate}

\section*{反事实分析}

\begin{limitbox}[若时间平移对称性破缺（能量不守恒）]
  \begin{itemize}
    \item 若 $\partial L/\partial t \neq 0$（系统 Lagragian 显含时间）：$dH/dt = -\partial L/\partial t \neq 0$——能量不守恒
    \item \textbf{现实例子}：膨胀宇宙的度规显含时间——宇宙总能量无全局守恒定义（但 $\nabla_\mu T^{\mu\nu}=0$ 仍成立）
    \item 若能量不守恒在所有尺度上都成立——永动机可能——宇宙不存在稳定物质结构
  \end{itemize}
\end{limitbox}

\begin{limitbox}[若电荷不守恒（$U(1)$ 对称性破缺）]
  \begin{itemize}
    \item 电子可衰变 $e \to \nu\gamma$——所有原子不稳定
    \item 电流连续性方程 $\partial_\mu J^\mu=0$ 破缺——Maxwell 方程需要修改
    \item 实验：电子寿命 $> 6.6\times10^{28}$ 年（Borexino）——$U(1)$ 对称性保护得极其牢固
  \end{itemize}
\end{limitbox}
