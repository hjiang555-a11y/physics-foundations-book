\section{概述}
R2 最优雅的推论是 Noether 定理（1918）：作用量的每个连续对称性对应一个守恒量。这不是巧合——它是变分法的数学必然。

\section{能量守恒}

\begin{lawbox}[Energy Conservation]
\[
\frac{dE}{dt} = 0 \quad (\text{孤立系统}), \qquad \partial_\mu T^{\mu\nu} = 0 \quad (\text{局域形式})
\]
\textbf{前提}：Noether 定理 + 时间平移对称
\end{lawbox}

\begin{derivationbox}[推导]
  若拉格朗日量 $L$ 不显含时间（$\partial L/\partial t = 0$），则时间平移是作用量的对称性。
  Noether 荷为哈密顿量：
  \[
  H = \sum_i p_i \dot{q}_i - L
  \]
  由 Euler-Lagrange 方程，$dH/dt = 0$。定义 $E \equiv H$ 即得能量守恒。

  \textbf{直觉}：今天和明天物理定律相同——因此能量不随时间改变。
\end{derivationbox}

\section{动量守恒}

\begin{lawbox}[Momentum Conservation]
\[
\frac{d\mathbf{P}}{dt} = 0 \quad (\text{孤立系统}), \qquad \partial_\mu T^{\mu i} = 0
\]
\textbf{前提}：Noether 定理 + 空间平移对称
\end{lawbox}

\begin{derivationbox}[推导]
  若 $L$ 在 $\mathbf{r} \to \mathbf{r} + \boldsymbol{\epsilon}$ 下不变，Noether 荷为总动量：
  \[
  \mathbf{P} = \sum_i m_i \mathbf{v}_i
  \]
  空间均匀 $\implies$ 动量守恒。这也给出了牛顿第三定律。

  \textbf{直觉}：北京和纽约物理定律相同——因此动量不随位置改变。
\end{derivationbox}

\section{角动量守恒}

\begin{lawbox}[Angular Momentum Conservation]
\[
\frac{d\mathbf{L}}{dt} = 0 \quad (\text{有心力场})
\]
\textbf{前提}：Noether 定理 + 旋转对称
\end{lawbox}

\begin{derivationbox}[推导]
  若 $L$ 在旋转下不变，Noether 荷为总角动量：
  \[
  \mathbf{L} = \sum_i \mathbf{r}_i \times \mathbf{p}_i
  \]
  有心力（$\mathbf{F} \parallel \mathbf{r}$）下 $\boldsymbol{\tau} = \mathbf{r}\times\mathbf{F} = 0 \implies d\mathbf{L}/dt = 0$。

  \textbf{直觉}：朝北和朝东物理定律相同——因此角动量守恒。
\end{derivationbox}

\section{电荷守恒}

\begin{lawbox}[Charge Conservation]
\[
\frac{dQ}{dt} = 0, \qquad \partial_\mu J^\mu = 0
\]
\textbf{前提}：Noether 定理 + $U(1)$ 规范对称（R3）
\end{lawbox}

\begin{derivationbox}[推导]
  Dirac 拉格朗日量 $\mathcal{L} = \bar{\psi}(i\gamma^\mu\partial_\mu - m)\psi$ 在全局 $U(1)$ 下不变。
  Noether 流：
  \[
  J^\mu = e\,\bar{\psi}\gamma^\mu\psi
  \]
  满足 $\partial_\mu J^\mu = 0$。空间积分：
  \[
  Q = \int J^0 d^3x, \qquad \frac{dQ}{dt} = -\oint \mathbf{J}\cdot d\mathbf{A} = 0
  \]
  \textbf{直觉}：你可以创造电子-正电子对（净电荷零），但总电荷永远不变。
\end{derivationbox}

\section{守恒律与连续性方程}

\begin{lawbox}[连续性方程 / Continuity Equation]
\[
\frac{\partial\rho}{\partial t} + \nabla\cdot\mathbf{J} = 0
\]
\end{lawbox}

\section{小结}

Noether 定理是物理学中最深刻的联系之一：\textbf{守恒不是偶然的巧合——它是宇宙几何对称性的直接表现}。

\begin{enumerate}
  \item 时间均匀 $\to$ 能量守恒
  \item 空间均匀 $\to$ 动量守恒
  \item 空间各向同性 $\to$ 角动量守恒
  \item 规范对称 $\to$ 电荷守恒
  \item 所有守恒律都有局域连续性方程 $\partial_\mu J^\mu = 0$
\end{enumerate}

如果明天物理定律突然改变了（时间平移对称破缺），能量就不再守恒。这是 R2 最优雅的遗产。
