\begin{insightbox}[规范不变性如何逼出电磁力]
  考虑一个复标量场 $\psi(x)$ 的拉格朗日量 $\mathcal{L} = |\partial_\mu\psi|^2 - m^2|\psi|^2$。物理可观测的是 $|\psi|^2$（概率密度），不是 $\psi$ 本身的相位。因此，如果你在每个时空点 $x$ 上独立地旋转 $\psi$ 的相位——$\psi(x) \to e^{i\alpha(x)}\psi(x)$——物理预言不应改变。

  但有一个问题。导数 $\partial_\mu\psi$ 在局域相位旋转下不协变地变换：
  \[
  \partial_\mu(e^{i\alpha(x)}\psi) = e^{i\alpha(x)}[\partial_\mu\psi + i(\partial_\mu\alpha)\psi] \neq e^{i\alpha(x)}\partial_\mu\psi
  \]
  多出来的 $i(\partial_\mu\alpha)\psi$ 项破坏了不变性——除非你引入一个补偿场 $A_\mu$ 来消掉它。这个$A_\mu$ 就是电磁规范势，它对应的场强 $F_{\mu\nu}$ 就是电场和磁场。

  \textbf{对称性要求（R3）→ 补偿场必然存在 → 规范场 = 光子 → 电磁力。}
  这不是实验发现——这是数学推导的\textbf{要求}。如果你的理论有局域 $U(1)$ 对称性，你就必然有电磁学。没有选择余地。
\end{insightbox}

\section{物理直觉：场的语言}

在 Maxwell 之前，电磁学是一堆不相关的经验定律——Coulomb 定律（电荷间的力）、Biot-Savart 定律（电流产生磁场）、Faraday 定律（变化的磁通量产生电动势）。Maxwell 1873 年做的，是把它们统一在\textbf{场的语言}中。

关键的概念跃迁是：\textbf{力不是瞬时的，而是通过场传播的。} 电荷不直接对远处的另一个电荷施加力——它先在自己周围的空间中产生一个电场 $\mathbf{E}$，这个电场再对位于其中的电荷施加力。场是真实存在的物理实体，携带动量、能量和角动量。

这就解释了为什么规范不变性如此根本：$A_\mu$ 是场的语言——它不直接可测（可测的是场强 $F_{\mu\nu}$），但它是场的完整数学描述。规范自由度（$A_\mu \to A_\mu + \partial_\mu\alpha$ 不改变任何可测物理量）恰好反映了：场有冗余描述，但物理内容是唯一的。

\section{推导：从 $U(1)$ 规范对称性到 Maxwell 方程}

\begin{derivationbox}[G4 基础 — 规范场框架]
  \textbf{Step 1} — U(1) 规范场强：$F_{\mu\nu} \equiv \partial_\mu A_\nu - \partial_\nu A_\mu$
  \hfill\texttt{[N]} $A_\mu$ 来自 R3 的规范补偿场要求

  \textbf{Step 2} — E, B 的场强表示：
  $F_{0i} = E_i/c$，$F_{ij} = -\varepsilon_{ijk}B_k$（度规 $(+,-,-,-)$）

  \textbf{Step 3} — Maxwell Lagrangian（SI 单位）：
  $\mathcal{L} = -\frac{1}{4\mu_0}F_{\mu\nu}F^{\mu\nu} - J^\mu A_\mu$
  \hfill\texttt{[N]} 满足 R1 (Lorentz 不变) 和 R3 (规范不变)

  \textbf{Step 4} — Euler-Lagrange 对 $A_\mu$ 变分：
  $\partial_\nu[\partial\mathcal{L}/\partial(\partial_\nu A_\mu)] - \partial\mathcal{L}/\partial A_\mu = 0$

  \textbf{Step 5} — 计算变分：$\partial\mathcal{L}/\partial(\partial_\nu A_\mu) = -\frac{1}{\mu_0}F^{\nu\mu}$，$\partial\mathcal{L}/\partial A_\mu = -J^\mu$

  \textbf{Step 6} — 协变 Maxwell 方程：$\partial_\nu F^{\nu\mu} = \mu_0 J^\mu$

  \textbf{Step 7} — $\mu=0$ $\Rightarrow$ Gauss 电定律：
  $\partial_i F^{i0} = \mu_0 J^0 \Rightarrow \partial_i(E_i/c) = \mu_0(c\rho_q) \Rightarrow \nabla\cdot\mathbf{E} = \rho_q/\varepsilon_0$
  \hfill\textit{注：度规 $(+,-,-,-)$ 下 $F^{i0} = F_{0i} = E_i/c$，\textbf{非} $-E_i/c$——两个负号抵消}

  \textbf{Step 8} — $\mu=i$ $\Rightarrow$ Amp\`{e}re-Maxwell 定律

  \textbf{Step 9} — Bianchi 恒等式（$F = dA$ 的几何必然）：
  $\partial_\lambda F_{\mu\nu} + \partial_\mu F_{\nu\lambda} + \partial_\nu F_{\lambda\mu} \equiv 0$
  \hfill\texttt{[N]} $A_\mu \in C^2$

  \textbf{Step 10} — Bianchi $\Rightarrow$ Gauss 磁：$\nabla\cdot\mathbf{B} = 0$

  \textbf{Step 11} — Bianchi $\Rightarrow$ Faraday：$\nabla\times\mathbf{E} = -\partial\mathbf{B}/\partial t$
\end{derivationbox}

\subsection{度量符号的重要警告}

在 $(+,-,-,-)$ 度规下，$F^{i0}$ 的计算需要格外小心：
\[
F^{i0} = g^{i\alpha}g^{0\beta}F_{\alpha\beta} = g^{ii}g^{00}F_{i0} = (-1)(1)F_{i0} = -F_{i0} = -(-F_{0i}) = F_{0i} = E_i/c
\]
\textbf{两个负号抵消}（度规下降指标 + 反对称性），最终结果与非相对论记号一致。这是迭代过程中发现的最隐蔽错误。

\subsection{Maxwell 方程的深层结构}

\keypoint{Maxwell 四个方程分为两类，来源截然不同。}

\textbf{第一类：来自动力学的方程}（Gauss 电 + Amp\`{e}re-Maxwell）——它们来自对 $A_\mu$ 的 Euler-Lagrange 变分，编码了规范场如何响应物质源 $J^\mu$。这两条是$\partial_\nu F^{\nu\mu} = \mu_0 J^\mu$ 的 $\mu=0$ 和 $\mu=i$ 分量。

\textbf{第二类：来自几何必然性的方程}（Gauss 磁 + Faraday）——它们是 Bianchi 恒等式 $\partial_{[\lambda}F_{\mu\nu]}=0$ 的分量形式，纯粹来自 $F = dA$ 的数学结构。$F$ 是闭形式（$dF = ddA \equiv 0$），这是微分几何的恒等式——不需要任何物理输入。

\textbf{这个分类含义深刻}：Gauss 磁 $\nabla\cdot\mathbf{B}=0$ 不是物理定律——它是"$\mathbf{B}$ 是 $A$ 的旋度"这个\textbf{定义}的必然推论。如果你接受 $\mathbf{B} \equiv \nabla\times\mathbf{A}$，那么 $\nabla\cdot(\nabla\times\mathbf{A}) \equiv 0$ 是矢量分析恒等式。同样，Faraday 定律是 $\partial\mathbf{B}/\partial t = \nabla\times(\partial\mathbf{A}/\partial t)$ 和 $\mathbf{E} = -\nabla\phi - \partial\mathbf{A}/\partial t$ 的结合。

所以 Maxwell 方程组实际上只有\textbf{两条独立的物理方程}（从变分原理给出），另外两条是几何恒等式。这是物理学中"规范冗余导致约束"的典型例子。

\bridge{规范不变性 $\to$ 引入 $A_\mu$ $\to$ $F_{\mu\nu} = \partial_\mu A_\nu - \partial_\nu A_\mu$ \\ $\xrightarrow{\text{Euler-Lagrange}}$ $\partial_\nu F^{\nu\mu} = \mu_0 J^\mu$ $\xrightarrow{\text{分量分解}}$ Gauss 电 + Amp\`ere-Maxwell \\ $\xrightarrow{\text{Bianchi}}$ $\partial_\lambda F_{\mu\nu} + \partial_\mu F_{\nu\lambda} + \partial_\nu F_{\lambda\mu}=0$ $\xrightarrow{\text{分量分解}}$ Gauss 磁 + Faraday}{$U(1)$ 对称性 + Lorentz 不变性 + 最小作用量 $\Rightarrow$ \textbf{电磁学的全部内容} \\ 不可能有其他结果——数学完全决定了形式 \\ 仅有的自由参数 $\varepsilon_0$（或等价的 $\mu_0$ 和 $c = 1/\sqrt{\varepsilon_0\mu_0}$）由实验确定}

\section{Lorentz 力定律}

\subsection{物理直觉}

电场推着电荷沿电场线方向运动；磁场却\textbf{不改变}电荷的速度大小——它只改变方向。这是因为磁力 $\mathbf{F} = q\mathbf{v}\times\mathbf{B}$ 总是垂直于速度方向，所以 $\mathbf{F}\cdot\mathbf{v} = 0$——磁场不做功！

这解释了为什么磁铁可以吸住冰箱上的便签——但吸力不是磁场直接做的功。是磁场改变了铁原子中电子的轨道运动，改变了系统的磁能，而磁能的梯度产生了合力。细节在微观，宏观表现却是"磁力"。

\keypoint{磁场不做功（$\mathbf{F}_B\cdot d\mathbf{r}=0$）。所有"磁力做功"的现象——电机转动、磁铁吸物——实际上是磁场改变了系统的能量配置，然后由其他力（如 Lorenz 力的电场部分或量子交换力）做功。}

\begin{lawbox}[Lorentz 力定律]
  $\mathbf{F} = q(\mathbf{E} + \mathbf{v}\times\mathbf{B})$
  \quad\textbf{前提}：R3（最小耦合）, R1\quad\textbf{推导号}：\texttt{deriv.lorentz\_force}
\end{lawbox}

\noindent\textbf{变量}：电荷 $q$ [T$\cdot$I], 速度 $\mathbf{v}$ [L$\cdot$T$^{-1}$], 标势 $\phi$, 矢势 $\mathbf{A}$。

\begin{derivationbox}[推导：最小耦合到 Lorentz 力]
  \textbf{Step 1} — 最小耦合 Lagrangian：
  $L = \frac{1}{2}m|\mathbf{v}|^2 - q\phi + q\mathbf{v}\cdot\mathbf{A}$
  \hfill\texttt{[N]} $p \to p - qA$（R3 的最小耦合）

  \textbf{Step 2} — 正则动量：$\mathbf{p}_{\text{can}} = \partial L/\partial\mathbf{v} = m\mathbf{v} + q\mathbf{A}$

  \textbf{Step 3} — Euler-Lagrange：$d\mathbf{p}_{\text{can}}/dt = \partial L/\partial\mathbf{r}$

  \textbf{Step 4} — 全导数展开：$d(m\mathbf{v})/dt + q\,d\mathbf{A}/dt = d(m\mathbf{v})/dt + q(\partial\mathbf{A}/\partial t + (\mathbf{v}\cdot\nabla)\mathbf{A})$

  \textbf{Step 5} — $\partial L/\partial\mathbf{r}$ 展开（矢量恒等式 $\nabla(\mathbf{v}\cdot\mathbf{A}) = (\mathbf{v}\cdot\nabla)\mathbf{A} + \mathbf{v}\times(\nabla\times\mathbf{A})$）

  \textbf{Step 6} — 代入化简，$(\mathbf{v}\cdot\nabla)\mathbf{A}$ 项抵消 $\implies \mathbf{F} = q(-\nabla\phi - \partial\mathbf{A}/\partial t) + q\mathbf{v}\times(\nabla\times\mathbf{A}) = q(\mathbf{E} + \mathbf{v}\times\mathbf{B})$
\end{derivationbox}

\begin{oombox}[Lorentz 力的实际大小]
  \begin{itemize}
    \item \textbf{电子在 1T 磁场中以 $10^6$ m/s 运动}：$F = evB = 1.6\times 10^{-13}$ N。加速度 $a = F/m_e = 1.8\times 10^{17}$ m/s$^2$（非常巨大！）
    \item \textbf{回旋半径}：$r = mv/(eB) = 5.7\,\mu$m——电子在微米尺度转圈
    \item \textbf{同步辐射}：电子弯曲时发射电磁辐射。LHC 中质子能量 7 TeV，回旋半径 $\sim 4.3$ km（$B=8.3$ T），同步辐射功率 $\sim$ 几 keV/圈
    \item \textbf{极光}：太阳风电子沿地球磁力线螺旋运动，在极区上空的稀薄大气中碰撞激发氮氧原子——绿色（氧 557.7 nm）、红色（氧 630 nm）、蓝色（氮）
  \end{itemize}
\end{oombox}

\precisionbox[精度等级]
  Lorentz 力定律：\precexact——它是最小耦合原理的数学推论，不依赖近似。但在强场（Schwinger 极限 $E_{\text{crit}} = m_e^2c^3/(e\hbar) \approx 1.3\times 10^{18}$ V/m）时，QED 非线性效应（真空双折射、光子-光子散射）修正 Lorentz 力。
\endprecisionbox}

\section{电磁波方程}

\subsection{物理直觉：Maxwell 最伟大的预言}

1865 年，Maxwell 把他的四个方程联立求解真空中的情况。他做了一个纯数学的推导——取旋度、代入、简化——然后得到了一个波动方程。波的传播速度是：

\[
c_{\text{theory}} = \frac{1}{\sqrt{\varepsilon_0\mu_0}}
\]

他把当时已知的 $\varepsilon_0$ 和 $\mu_0$ 值代入……得到的速度正好是已知的光速！

这不是巧合。Maxwell 随即宣布：\textbf{光就是电磁波。} 这是人类历史上第一次，一个全新的物理现象（电磁波）完全从理论推导中\textbf{预言}出来，然后才被实验证实（Hertz 1887）。Heinrich Hertz 后来评论："Maxwell 的理论就是他的方程组。"

\keypoint{Maxwell 没有"发现"电磁波——他\textbf{推导}出了它。他从纯数学对称性（Amp\`ere 定律加上位移电流项）出发，得到了波的方程，发现波速恰为光速。这是理论物理最辉煌的时刻之一——数学一致性的要求预言了一个全新的物理现象。}

\begin{lawbox}[电磁波方程]
  $\nabla^2\mathbf{E} = \frac{1}{c^2}\frac{\partial^2\mathbf{E}}{\partial t^2}$，$\nabla^2\mathbf{B} = \frac{1}{c^2}\frac{\partial^2\mathbf{B}}{\partial t^2}$，$c = 1/\sqrt{\varepsilon_0\mu_0}$
  \quad\textbf{前提}：Maxwell 四方程（真空）
\end{lawbox}

\begin{derivationbox}[推导]
  \textbf{Step 1} — 真空 Maxwell 方程（$\rho_q=0$, $\mathbf{J}=0$）

  \textbf{Step 2} — 对 Faraday 取旋度：$\nabla\times(\nabla\times\mathbf{E}) = -\frac{\partial}{\partial t}(\nabla\times\mathbf{B})$
  \texttt{[N]} $\mathbf{E},\mathbf{B} \in C^2$

  \textbf{Step 3} — 矢量恒等式 + $\nabla\cdot\mathbf{E}=0$：$\nabla\times(\nabla\times\mathbf{E}) = \nabla(\nabla\cdot\mathbf{E}) - \nabla^2\mathbf{E} = -\nabla^2\mathbf{E}$

  \textbf{Step 4} — 右边代入 Amp\`{e}re-Maxwell：$-\frac{\partial}{\partial t}(\mu_0\varepsilon_0\frac{\partial\mathbf{E}}{\partial t}) = -\mu_0\varepsilon_0\frac{\partial^2\mathbf{E}}{\partial t^2}$

  \textbf{Step 5} — $\nabla^2\mathbf{E} = \mu_0\varepsilon_0\frac{\partial^2\mathbf{E}}{\partial t^2} = \frac{1}{c^2}\frac{\partial^2\mathbf{E}}{\partial t^2}$，其中 $c \equiv 1/\sqrt{\varepsilon_0\mu_0}$
\end{derivationbox}

\textbf{必要条件}：Maxwell 四方程 [N], 真空 [N] ($\rho_q=0,\mathbf{J}=0$), $\mathbf{E},\mathbf{B} \in C^2$ [N]。

\textbf{充分条件}：真空 Maxwell 方程 $\Rightarrow$ $\mathbf{E},\mathbf{B}$ 满足波动方程，波速 $c = 1/\sqrt{\varepsilon_0\mu_0}$。\texttt{reversible: false}（波动方程丢失了 $\nabla\cdot\mathbf{E}=0$ 和 $\nabla\cdot\mathbf{B}=0$ 约束）。

\bridge{$\nabla\times(\nabla\times\mathbf{E}) = \nabla(\nabla\cdot\mathbf{E}) - \nabla^2\mathbf{E}$ \\ $= -\nabla^2\mathbf{E}$（因为 $\nabla\cdot\mathbf{E}=0$ 在真空中）\\ $= -\frac{\partial}{\partial t}(\nabla\times\mathbf{B}) = -\mu_0\varepsilon_0\frac{\partial^2\mathbf{E}}{\partial t^2}$}{真空中的 Maxwell 方程 $\to$ 必然存在以速度 $c$ 传播的横波 \\ 这个 $c$ 恰好等于光速——因此\textbf{光就是电磁波}}

\section{本章小结：一个完整的逻辑链条}

从 R3（局域规范不变性）到可观测的电磁现象，整个链条是：

\[
\boxed{\text{R3: $U(1)$ 局域规范对称性}}
\xrightarrow{\text{补偿场需求}}
\boxed{A_\mu \text{ 必然存在}}
\xrightarrow{\text{R1: Lorentz 不变性}}
\boxed{\mathcal{L} = -\frac{1}{4\mu_0}F_{\mu\nu}F^{\mu\nu} - J^\mu A_\mu}
\xrightarrow{\text{R2: }\delta S = 0}
\boxed{\partial_\nu F^{\nu\mu} = \mu_0 J^\mu}
\xrightarrow{\text{分量分解}}
\boxed{\text{Maxwell 四方程}}
\xrightarrow{\text{联立求解}}
\boxed{\text{电磁波：}c = 1/\sqrt{\varepsilon_0\mu_0} = \text{光速}}
\]

\textbf{这个链条中的每一步都是逻辑必然的。} 如果你接受 R1 + R2 + R3 + $U(1)$ 规范群，你就必然得到 Maxwell 方程组和电磁波的存在。没有任何自由参数选择的空间——除了 $\varepsilon_0$（或等价的 $c$ 和 $\mu_0$）需要实验测定。

物理学的这种"从数学一致性要求逼出物理理论"的特点，在电磁学中表现得最为干净利落。这也是为什么规范理论成为现代物理学核心框架的原因——它告诉你：\textbf{对称性决定相互作用。}

\keypoint{从 R3 到 Maxwell 的 11 步推导中，每一步都是逻辑必然的。如果你接受 $U(1)$ 规范对称性 + Lorentz 不变性 + 最小作用量，Maxwell 方程就是你必然得到的电磁理论。没有任何其他选择——数学已经替你做了唯一的决定。}
\section*{反事实分析}

\begin{limitbox}[若无位移电流项]
  电荷不守恒 $\nabla\cdot\mathbf{J} \neq 0$；无电磁波——Faraday+Amp\`ere 联立得不到波动方程；光速无法从 $\varepsilon_0,\mu_0$ 推导。
\end{limitbox}

\begin{limitbox}[若磁单极子存在]
  Maxwell 方程完全对称。Dirac (1931) 证明即使一个磁单极子也解释电荷量子化（$eg = n\hbar/2$）。至今未发现——flux upper limit $< 10^{-16}$ cm$^{-2}$s$^{-1}$sr$^{-1}$。
\end{limitbox}

\section*{习题}

\begin{enumerate}
  \item \textbf{[简单]} 从 $F_{\mu\nu} = \partial_\mu A_\nu - \partial_\nu A_\mu$ 验证 Bianchi 恒等式。写出 $(\lambda,\mu,\nu)=(0,1,2),(0,1,3)$ 对应的三维矢量方程。
  \item \textbf{[简单]} 从 $B_\mu = \partial_\mu\alpha(x)$ 证明 $F_{\mu\nu} \equiv 0$——这对应什么物理情况？
  \item \textbf{[中等]} 用 $\mathcal{L} = -\frac{1}{4\mu_0}F_{\mu\nu}F^{\mu\nu} - J^\mu A_\mu$ 对 $A_\mu$ 做 Euler-Lagrange 变分，导出 $\partial_\nu F^{\nu\mu} = \mu_0 J^\mu$。
  \item \textbf{[中等]} 在 Coulomb 规范（$\nabla\cdot\mathbf{A}=0$）下化简 Maxwell 方程为 $\nabla^2\phi = -\rho_q/\varepsilon_0$ 和 $\Box\mathbf{A} = \mu_0\mathbf{J} - \nabla(\partial\phi/\partial t)/c^2$。
  \item \textbf{[较难]} 从 Lorentz 力 Lagrangian 写出正则动量。证明均匀磁场中 $L_z + \frac{qB}{2}(x^2+y^2)$ 守恒。
  \item \textbf{[挑战]} 两运动点电荷间电磁力：$v\ll c$ 时近似满足 Newton 第三定律；$v\sim c$ 时场携带动量使第三定律破缺。估算 $v=0.1c$ 时偏离大小。
\end{enumerate}
