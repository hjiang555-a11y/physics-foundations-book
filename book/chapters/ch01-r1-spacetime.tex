\section{总说：宇宙的舞台}

物理学研究的是"什么在什么上面发生"。在问"什么规律"之前，我们必须先回答一个更根本的问题：\textbf{发生的舞台是什么样子}？R1 回答的就是这个问题。

\subsection{核心图像：光速的绝对性}

想象你站在站台上，一列火车以 $u = 100$ km/h 的速度驶过。有人在火车上以 $v' = 10$ km/h 向前扔球。你理所当然地预期球的速度是 $u + v' = 110$ km/h。\textbf{我们日常的全部直觉都建立在"速度可以直接相加"这一假设上}。

1905 年，爱因斯坦用一句话颠覆了这个直觉世界：\textbf{真空中的光速与光源的运动无关}。不论你站在站台上还是在飞驰的火车上，激光束的速度始终是同一个数——$c \approx 3 \times 10^8$ m/s。这不是一个近似的观测结果，而是 Maxwell 电磁理论的逻辑推论：电磁波的速度 $c = 1/\sqrt{\varepsilon_0\mu_0}$ 仅由真空本身的电磁性质决定，与谁在测量无关。

光锥将时空分为三个互不相通的区域：(1) \textbf{未来光锥内部}——我能影响的事件；(2) \textbf{过去光锥内部}——能影响我的事件；(3) \textbf{类空区域}（"别处"）——与我\textbf{绝对无关}的事件。这个划分是绝对的——不随观察者的运动状态而改变。换言之：\textbf{因果不是"视角问题"，它是时空结构的几何性质}。

\subsubsection{时间不再是绝对的}

光速不变性的第二个后果更为反直觉：\textbf{同时性依赖于参考系}。考虑一列以速度 $v$ 运动的火车。当火车正中间的一个灯泡闪亮时，火车上的人认为光同时到达车头和车尾。但站台上的人看到：车尾迎着光运动，车头远离光运动——光到达车尾的时刻\textbf{早于}光到达车头的时刻。

这个思想实验揭示了一个深刻的事实：\textbf{不存在一个"上帝视角"的绝对钟表}。"现在"是相对的——你的"现在"切片和我的"现在"切片可能不同。这正是 Lorentz 变换的物理本质。

\subsubsection{广义相对论的一个图像}

等效原理告诉我们 $m_i = m_g$。想象一个无窗的电梯：电梯里的人无法通过任何实验区分"电梯静止在地球表面"和"电梯在远离一切引力的太空中以 $g = 9.8$ m/s\textsuperscript{2} 向上加速"。既然加速参考系与弯曲坐标系等价（你的加速使得遥远恒星的光线在你看来弯曲），引力就等同于时空的几何弯曲。

用一个简单的比喻：想象一个绷紧的橡胶膜（代表时空）。放一个保龄球上去——橡胶膜凹陷。现在在凹陷处滚一个小球——小球不会沿直线走，而是绕着凹陷弯曲。\textbf{在 Newton 的语言里，我们说"保龄球的引力拉住了小球"。在 Einstein 的语言里，我们说"保龄球弯曲了时空，小球沿着弯曲时空的最直路径（测地线）运动"}。前者是力的语言，后者是几何的语言——等效原理允许我们在两者之间自由切换。

\subsection{为什么这条原则必不可少}

试想一个没有光速上限的宇宙。在这样一个宇宙里：
\begin{itemize}
  \item 信息可以瞬间传播到任意远处。这意味着你可以收到来自未来的信号——因果关系崩溃。
  \item 粒子可以以任意速度运动，无法形成稳定的原子结构。
  \item 电磁波不会存在（因为 c 是电磁波速度，而 c 不是一个特殊值）。
\end{itemize}

\noindent 正是因为光速有限且绝对，我们的宇宙才有了清晰的因果结构——过去、现在和未来的严格划分。这是全部现代物理（从狭义相对论到量子场论）的逻辑起点。

\subsection{引力为什么是时空弯曲}

R1 还包含一条独立的深刻洞见：\textbf{惯性质量和引力质量严格相等}（等效原理）。

在地球上，你感受到的重力（引力质量）和物体的"抗拒加速的本性"（惯性质量）由同一个 $m$ 描述。从伽利略的比萨斜塔实验到 Eötvös 的高精度扭摆测量（精度达 $10^{-15}$ 量级），这个等式从未被打破。

等效原理的后果是：在局部范围内，你无法区分"静止在地球表面"和"在远离引力场的太空中以 $g=9.8$ m/s² 加速"。既然引力等价于加速参考系，而加速参考系等价于弯曲坐标系，那么——\textbf{引力就是时空几何}。这正是广义相对论的核心思想。

\section{严格陈述}

\subsection{R1 包含三个逻辑上独立的子原则}

\begin{principlebox}[R1 — 因果时空结构]
  \textbf{(1a)  最大传播速度}（核心公设）：
  真空中任何信号、能量或因果影响的传播速度不超过 $c$。$c$ 在所有惯性系中取相同数值——它是普适常数，不是特定参考系的属性。

  \textbf{(1b)  Lorentz 因果结构}（与 1a 等价）：
  时空间隔 $ds^2 = g_{\mu\nu}dx^\mu dx^\nu$ 的符号将事件对分为三类——类时（$ds^2 > 0$，可因果连接）、类光（$ds^2 = 0$，仅光速信号）、类空（$ds^2 < 0$，无因果联系）。
  因为 $ds^2 = c^2dt^2 - |d\mathbf{r}|^2$，类空 $\iff |d\mathbf{r}|/dt > c$——所以"无超光速信号"(1a) 与"类空事件无因果联系"(1b) 是\textbf{同一事实的两种表述}。

  \textbf{(1c)  等效原理}（独立公设，与 1a/1b \textbf{逻辑独立}）：
  惯性质量严格等于引力质量（$m_i = m_g$）。局域上，引力场与加速参考系不可区分。此原理与光速不变无逻辑蕴含关系——$m_i=m_g$ 是引力可被几何化的必要桥梁。等效原理 + 广义协变性 $\Rightarrow$ 引力即时空弯曲 $\Rightarrow$ 广义相对论。

  \textbf{广义协变性}：物理定律在所有坐标系中取相同形式。
  这是 R1 内嵌的结构原则：一旦等效原理将引力升格为时空几何的问题，广义协变性就是自然的数学要求。
\end{principlebox}

\section{数学表达式}

\subsection{平直时空：Minkowski 度规}

在无引力的局域惯性系中，时空取最简单的形式——Minkowski 度规：

\begin{equation}
ds^2 = c^2 dt^2 - (dx^1)^2 - (dx^2)^2 - (dx^3)^2
\end{equation}

\noindent 这是狭义相对论的全部几何基础。符号差 $(+,-,-,-)$ 意味着时间方向贡献正的线元，三个空间方向贡献负的线元。这种符号差（一个正号，三个负号）称为 Lorentz 符号差，它是因果结构存在的前提。

\subsection{弯曲时空：一般坐标}

在存在引力的情况下，度规不再是常数——它成为位置的函数 $g_{\mu\nu}(x)$：

\begin{equation}
ds^2 = g_{\mu\nu}(x)\, dx^\mu dx^\nu
\end{equation}

\noindent $g_{\mu\nu}$ 是一个 $4\times4$ 的对称张量，由于对称性（$g_{\mu\nu}=g_{\nu\mu}$），它有 $D(D+1)/2 = 10$ 个独立分量。在三维空间中引力场的自由度对应两个偏振态（因此引力波有两个偏振方向）。

\subsection{因果分类}

对任意两个事件 $A$ 和 $B$，定义时空间隔：
\begin{equation}
\Delta s^2 \equiv g_{\mu\nu} \Delta x^\mu \Delta x^\nu
\end{equation}

\noindent 按 $\Delta s^2$ 的符号，事件对分为三类：

\subsection{局域性约束}

因果结构还有一个重要推论：物理量 $\phi(x)$ 在时空点 $x$ 的值仅依赖于其在过去光锥 $J^-(x)$ 内的初始数据。这意味着物理影响不能从光锥之外传播进入——信息只能沿着类时或类光路径传播。

这一约束是量子场论中\textbf{微观因果性}的基础，也是自旋-统计定理的核心前提之一（见第9章 Pauli 不相容原理的推导）。

\subsection{等效原理的定量表述}

弱等效原理的定量表述是：

\begin{equation}
\frac{m_i}{m_g} = 1 \quad \text{（在实验精度 $10^{-15}$ 范围内精确成立）}
\end{equation}

\noindent 这一等式意味着：\textbf{任何物体的引力加速度与其成分无关}——在真空中，羽毛和铅球以相同的加速度下落。

Einstein 将其提升为物理原理：\textbf{在局域范围内，一个均匀引力场中的物理定律与在无引力场中均匀加速的参考系中的物理定律完全相同}。这是连接引力和时空几何的关键洞察。

\section{严谨推导}

在本节中，我们从 R1 的基本假设出发，展示其最重要的推论——Lorentz 变换、时间膨胀与长度收缩、以及因果分类——是如何被严格推导出来的。每条推导步骤标注必要性条件 \fbox{[N]} 和充分性条件 \fbox{[S]}。

\subsection{推导 1：从光速不变性到 Lorentz 变换}

\textbf{前提}：光速在任何惯性系中均为 $c$（R1-1a）。时空是均匀的各向同性的（R1 暗含的结构假设）。

\textbf{推导路径}：考虑两个惯性系 $S$（静止）和 $S'$（沿 $x$ 轴以恒定速度 $v$ 运动），它们在 $t = t' = 0$ 时坐标原点重合。

\textbf{步骤 1 — 线性变换假设}：\fbox{[N]} 时空的均匀性要求坐标变换是线性变换（否则平移不变性被破坏）：
\begin{equation}
x' = A x + B t, \quad t' = C x + D t
\end{equation}
\fbox{[S]} 给定时空均匀性，线性变换是最一般的形式。非线性项会引入绝对位置依赖，破坏平移不变性。

\textbf{步骤 2 — 相对性原理}：\fbox{[N]} $S'$ 系原点（$x' = 0$）在 $S$ 系中以速度 $v$ 运动，即 $x = vt$。代入 $x' = 0$：
\begin{equation}
0 = A(vt) + B t = (Av + B)t \quad\Rightarrow\quad B = -Av
\end{equation}
故 $x' = A(x - vt)$。\fbox{[S]} 这一步仅使用了"$S'$ 相对 $S$ 以速度 $v$ 运动"这一定义。

\textbf{步骤 3 — 光速不变性约束}：\fbox{[N]}\fbox{[S]} 在 $t = t' = 0$ 时从共同原点发出一个光信号。在两个系中光波前方程分别为：
\begin{subequations}
\begin{align}
x^2 &= c^2 t^2 \quad \text{（$S$ 系）} \\
x'^2 &= c^2 t'^2 \quad \text{（$S'$ 系）}
\end{align}
\end{subequations}
这里我们假设了空间各向同性（$y,z$ 方向同理）。注意此时写的是 \textbf{$x^2 = c^2 t^2$} 而非 $x = ct$——因为光可以沿 $+x$ 或 $-x$ 方向传播，两式平方后等价。

\textbf{步骤 4 — 求解变换系数}：将变换代入 $x'^2 - c^2 t'^2 = 0$，并要求在任何满足 $x^2 - c^2 t^2 = 0$ 的点上成立。由于时空是均匀各向同性的 \fbox{[N]}，这一关系必须对所有时空点成立（非仅对光锥上的点），即 \textbf{Lorentz 不变量的普适性}：
\begin{equation}
x'^2 - c^2 t'^2 = x^2 - c^2 t^2
\end{equation}
将 $x' = A(x - vt)$ 和 $t' = Cx + Dt$ 代入左边，与右边对比 $x^2$, $t^2$, $xt$ 各项系数，得到三个方程：
\begin{align}
A^2 - c^2 C^2 &= 1 \label{eq:coeff1} \\
A^2 v^2 - c^2 D^2 &= -c^2 \label{eq:coeff2} \\
-A^2 v - c^2 CD &= 0 \label{eq:coeff3}
\end{align}
\fbox{[S]} 这三个代数方程唯一确定了 $A$, $C$, $D$ 的值（正负号由 $v \to 0$ 时 $x' \to x$, $t' \to t$ 的边界条件确定）：
\begin{equation}
A = D = \gamma \equiv \frac{1}{\sqrt{1 - v^2/c^2}}, \quad C = -\frac{\gamma v}{c^2}
\end{equation}

\textbf{步骤 5 — 最终形式}：
\begin{equation}
\boxed{t' = \gamma\left(t - \frac{v}{c^2}x\right), \quad x' = \gamma(x - vt), \quad y' = y, \quad z' = z}
\end{equation}
\fbox{[S]} 这就是 Lorentz 变换。当 $v \ll c$ 时 $\gamma \to 1$，恢复 Galileo 变换。注意：这里推导中 \textbf{只用到了 R1-1a（光速不变）和时空均匀性}——这是 R1 单独就能给出的结论。

\subsection{推导 2：时间膨胀与长度收缩}

这两个效应是 Lorentz 变换的直接几何推论。考虑 Lorentz 变换的逆变换（将 $S$ 看作运动系）：
\begin{equation}
t = \gamma\left(t' + \frac{v}{c^2}x'\right), \quad x = \gamma(x' + vt')
\end{equation}

\textbf{时间膨胀}：\fbox{[S]} 在 $S'$ 系中静止的钟（$\Delta x' = 0$）经历 $\Delta t'$（固有时间）。在 $S$ 系中测量：
\begin{equation}
\Delta t = \gamma\left(\Delta t' + \frac{v}{c^2} \cdot 0\right) = \gamma\, \Delta t' > \Delta t'
\end{equation}
\fbox{[N]} $\gamma = 1/\sqrt{1-v^2/c^2} > 1$ 的前提是 $v < c$——这是 R1-1a 的要求。若 $v \ge c$，$\gamma$ 变为虚数或无穷大——变换失效。运动的钟走得慢。这是真正的物理效应，不是"视错觉"——$\mu$ 子在大气层中寿命的延长是直接实验证据。

\textbf{长度收缩}：\fbox{[S]} 在 $S$ 系中测量运动杆的长度——必须同时记录两端的位置（$\Delta t = 0$）。杆在 $S'$ 中静止，其固有长度为 $\Delta x' = L_0$。在 $S$ 中：
\begin{equation}
\Delta x' = \gamma(\Delta x - v\Delta t) = \gamma(\Delta x - 0) = \gamma L
\end{equation}
故 $L = L_0/\gamma < L_0$。运动尺缩短。\fbox{[N]} 这里的"同时性"（$\Delta t = 0$）是长度测量的定义——不同参考系中的长度测量对应不同的"同时切片"，这正体现了时间与空间的不可分割性。

\subsection{推导 3：因果分类的几何来源}

\textbf{步骤}：定义两个事件的时空间隔：
\begin{equation}
(\Delta s)^2 \equiv c^2(\Delta t)^2 - |\Delta\mathbf{r}|^2
\end{equation}
\fbox{[S]} 从 Lorentz 变换可直接验证：$(\Delta s')^2 = (\Delta s)^2$——间隔是 Lorentz 不变量。这是我们将事件分类为"有因果联系"和"无因果联系"的\textbf{不变量判据}。

按 $(\Delta s)^2$ 的符号：
\begin{itemize}
\item $(\Delta s)^2 > 0$（类时）：存在一个参考系，其中两事件发生在同一地点（仅时间先后不同）\fbox{[S]}——可因果联系。
\item $(\Delta s)^2 = 0$（类光）：只有光速信号可连接两事件\fbox{[S]}。
\item $(\Delta s)^2 < 0$（类空）：存在参考系中两事件同时发生\fbox{[S]}——但此时第三系中它们可能"时间倒序"！由于 R1-1a 禁止超光速信号，类空事件\textbf{在任何参考系中}都无法因果连接。
\end{itemize}

\textbf{为什么类空事件因果无关是绝对的？} \fbox{[N]} 假设事件 $A$ 和 $B$ 类空分离但 $A$ 能因果影响 $B$。则存在一个以速度 $v < c$ 传播的信号从 $A$ 到 $B$。但 Lorentz 变换下，类空间隔的"时间先后"随参考系改变——存在一个参考系中 $B$ 在 $A$ \textbf{之前}发生。这导致因果悖论（果先于因）。\fbox{[S]} 因此 R1-1a（无超光速信号）与"类空事件无因果联系"（R1-1b）是\textbf{逻辑等效的}。

\subsection{推导 4：等效原理的局域性}

弱等效原理 $m_i = m_g$ 的推论可以通过 Newton 极限清晰地展示：

\textbf{步骤 1}：\fbox{[N]} 在非相对论极限下，Newton 第二定律（由 R1+R2 导出，见第 7 章）给出 $\mathbf{F} = m_i \mathbf{a}$。

\textbf{步骤 2}：\fbox{[N]} 在均匀引力场 $\mathbf{g}$ 中，引力为 $\mathbf{F}_g = m_g \mathbf{g}$。

\textbf{步骤 3}：\fbox{[S]} 若 $m_i = m_g$，则 $\mathbf{a} = \mathbf{g}$——\textbf{加速度与物体质量、成分无关}。这是 Galileo 比萨斜塔实验的数学表达。\fbox{[S]} 所有物体在引力场中自由下落的加速度相同。

\textbf{步骤 4}：\fbox{[S]} $\mathbf{a} = \mathbf{g}$ 意味着在自由下落参考系中，引力被"消除"——局域上无法区分无引力和自由下落。这正是 Einstein 将引力几何化的逻辑起点。但注意 \fbox{[N]} 这仅在"局域"（潮汐力可忽略的足够小的区域内）严格成立——大尺度上潮汐力 = 时空曲率，可在自由下落系中探测到。

\section{成立条件 (Scope)}

\begin{itemize}
  \item \textbf{有效场论范围}：R1 在能量远低于 Planck 尺度（$E \ll E_P \approx 1.22 \times 10^{19}$ GeV）、距离远大于 Planck 长度（$l \gg l_P \approx 1.616 \times 10^{-35}$ m）的范围内精确成立。这是目前所有已确认物理实验的适用域。
  \item \textbf{狭义相对论}：$g_{\mu\nu} \rightarrow \eta_{\mu\nu}$（平直极限，忽略引力）。在此极限下 R1 约化为狭义相对论的全部运动学。
  \item \textbf{广义相对论}：$g_{\mu\nu}$ 升格为动力学场，由 Einstein 场方程（$\leftarrow$ R1 等效原理 + R2 最小作用量）决定其演化。
  \item \textbf{局域性}：等效原理仅在"局域"（时空的一个小邻域）内严格成立。在大的时空尺度上，潮汐力使引力场和匀加速参考系可以区分。
\end{itemize}

\section{本章小结}

R1 定义了物理世界的"舞台"——四维 Lorentz 流形，以及这个舞台上的因果规则。它的三个子组件支撑着两个不同的物理理论分支：(1a+1b) 支撑狭义相对论和量子场论的因果结构；(1c) 独立地支撑广义相对论——引力作为时空几何的理论。

R1 为后续所有推导提供了"时间坐标"、"空间坐标"、"惯性参考系"、"因果关系"等基本概念。没有 R1，后面的 R2-R6 将失去它们的定义域和操作空间。

在第 11 章和第 12 章中，我们将看到 R1 如何直接导出 Lorentz 变换和质能等价（这两条仅需 R1 本身），以及如何与 R2 结合导出 Einstein 场方程。

\section*{实验验证精度}

\begin{oombox}[R1 的实验验证——从 Michelson-Morley 到 LIGO]
  \begin{itemize}
    \item \textbf{光速各向同性}：Michelson-Morley (1887) 零结果。现代光学腔实验（Müller 2003）将 Lorentz 破缺约束到 $\Delta c/c < 10^{-18}$
    \item \textbf{光速恒定}：对所有频率的光，$c$ 相同——利用伽马射线暴到达时间测得的差异 $< 10^{-15}$
    \item \textbf{等效原理}：Eötvös 实验（1885-1909）验证 $m_i/m_g$ 在 $10^{-9}$ 内为 1。现代 MICROSCOPE 卫星 (2022) 将差异约束到 $< 10^{-15}$——这是科学史上最精确的零结果之一
    \item \textbf{Lorentz 不变性}：在 $10^{-28}$ 精度内无破缺——来自极高能宇宙射线（$E > 10^{19}$ eV）的到达时间测量
    \item \textbf{引力波速度} = $c$：GW170817 的光学对应体与引力波信号几乎同时到达（$\Delta t < 1.7$ s，距离 $40$ Mpc）——约束 $\Delta v/c < 10^{-15}$
    \item \textbf{为什么 $c$ 是定义常数}：SI 2019 将 $c$ 定义为精确值 $2.99792458\times10^8$ m/s，因为光速是宇宙的结构属性而非测量值
  \end{itemize}
\end{oombox}

  \end{itemize}

    \item 引力波仍有（作为场的波），但不再是时空度规的涟漪
    \item \textbf{现实}：$m_i/m_g$ 在 $10^{-15}$ 精度内为 1——等效原理是物理学中最精确的非平凡零结果
  \end{itemize}
