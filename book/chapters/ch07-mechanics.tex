\subsection{科普层：三百年后 $F = ma$ 仍未被推翻}

苹果落在 Newton 头上的故事是后人的浪漫化（实际是他在乡下避瘟疫时观察到的）。但 Newton 1687 年的真正成就远比"发现重力"深刻：他证明了\textbf{使苹果落地的力，与维持月球绕地的力，是同一种力}——并以 $F = G m_1 m_2 / r^2$ 量化了它。

\textbf{Newton 把 Kepler 的三定律从经验规律提升为推论}。Kepler (1609-1619) 通过 Tycho Brahe 的火星观测归纳出三条经验定律：椭圆轨道、面积速度恒定、周期与半长轴 3/2 次方成正比。Newton 证明：只要假设引力是 $1/r^2$ 的中心力，所有三条 Kepler 定律就是数学推论。这是物理学第一次成功地把"现象学规律"还原为"基础动力学定律"。

但 Newton 的 $F = ma$ 形式在数学上并非最优雅。1788 年，Lagrange 在《分析力学》中提出：用广义坐标 $q_i$ 和拉氏量 $L = T - V$，运动方程是 $\partial L/\partial q_i = d(\partial L/\partial \dot{q}_i)/dt$。这个形式有三大优势：
\begin{enumerate}
\item \textbf{坐标无关}：不必区分笛卡尔/极/球坐标，自动处理约束。
\item \textbf{对称性显式}：Noether 定理直接从 $L$ 的对称性得到守恒量。
\item \textbf{推广无碍}：场论、相对论、量子力学都使用同一框架。
\end{enumerate}

\textbf{现代物理教学普遍以 Lagrange 形式为主}，正是因为它把 R2（最少作用量）作为根基，而 $F = ma$ 退化为它的特殊推论。Newton 形式仍在工程和入门课程中广泛使用，因为它直接、直观；但在深层物理研究中，几乎所有先进理论都从拉氏量出发。

\subsection{核心图像：单摆的两副语言}

考虑长度 $\ell$、质量 $m$ 的单摆，重力 $g$。设摆角 $\theta$ 为广义坐标。

\textbf{Newton 视角}：在切向方向，重力分量 $-mg\sin\theta$ 提供加速度 $\ell\ddot\theta$，得：
\[
\ell\ddot\theta = -g\sin\theta \quad \Rightarrow \quad \ddot\theta + (g/\ell)\sin\theta = 0
\]

\textbf{Lagrange 视角}：动能 $T = \tfrac{1}{2}m\ell^2\dot\theta^2$，势能 $V = mg\ell(1-\cos\theta)$。
\[
L = T - V = \tfrac{1}{2}m\ell^2\dot\theta^2 - mg\ell(1-\cos\theta)
\]
Euler-Lagrange 方程：
\[
\frac{d}{dt}\frac{\partial L}{\partial \dot\theta} - \frac{\partial L}{\partial \theta} = m\ell^2\ddot\theta + mg\ell\sin\theta = 0
\]
即 $\ddot\theta + (g/\ell)\sin\theta = 0$。

两种语言得到相同方程。但 Lagrange 视角的优势在于：如果加上约束（双摆、绳长变化等），Newton 视角需要小心处理张力等内力，Lagrange 视角自动消去内力——这就是它在复杂系统中胜出的原因。

\paragraph{小角近似量级估算} 当 $\theta \ll 1$，$\sin\theta \approx \theta$，方程退化为 $\ddot\theta = -(g/\ell)\theta$，简谐运动周期 $T = 2\pi\sqrt{\ell/g}$。1 米单摆 $T \approx 2$ 秒，与日常经验一致。

\section{Euler-Lagrange 方程}
% ════════════════════════════════════════════════════════

\subsection{物理直觉}

一个球从斜坡上滚下，它走的是哪条路？经验告诉我们：不是最快的路，不是最短的路——而是"作用量"最小的路。作用量 $S = \int L\,dt$ 是拉格朗日量 $L = T - V$ 在时间上的累积。球必须在每个瞬间"平衡"两种倾向：动能喜欢大（球想滚得快），势能喜欢小（球想待在低处）。真实路径是这两种倾向在所有可能路径中的最优折中。

\textbf{数学结构}：这就是一个\textbf{泛函极值问题}——在所有连接起点和终点的可能路径中，找一条使积分 $S = \int L(q,\dot{q},t)dt$ 取极值的路径。这完全类似于"在所有连接两点的曲线中找最短的那条"（那是测地线问题），只不过被积函数从弧长元 $\sqrt{1+(dy/dx)^2}$ 换成了拉格朗日量 $L$。

Euler-Lagrange 方程就是这个泛函极值问题的解——它告诉你极值路径必须满足的微分方程。它之于变分问题，如同 $f'(x)=0$ 之于普通函数的极值问题。
% ════════════════════════════════════════════════════════

\begin{lawbox}[Euler-Lagrange 方程]
  \[
  \frac{d}{dt}\!\left( \frac{\partial L}{\partial \dot{q}_i} \right) - \frac{\partial L}{\partial q_i} = 0
  \]
  \textbf{前提}：R2（$\delta S = 0$）\quad \textbf{推导号}：\texttt{deriv.euler\_lagrange}
\end{lawbox}

\subsection{推导}

\begin{derivationbox}[推导：从最小作用量到 Euler-Lagrange 方程]
  \textbf{Step 1} — 作用量定义：
  $S[q] = \int_{t_1}^{t_2} L(q,\dot{q},t)\,dt$
  \hfill\textit{(R2)}

  \textbf{Step 2} — 路径变分，端点固定：
  $\delta S = \int [L(q+\delta q, \dot{q}+\delta\dot{q},t) - L(q,\dot{q},t)]\,dt$
  \hfill\texttt{[N]} $\delta q(t_1)=\delta q(t_2)=0$

  \textbf{Step 3} — Taylor 展开至一阶：
  $\delta S = \sum_i \int [\frac{\partial L}{\partial q_i}\delta q_i + \frac{\partial L}{\partial \dot{q}_i}\delta\dot{q}_i]\,dt$
  \hfill\texttt{[N]} $L \in C^2$

  \textbf{Step 4} — 分部积分第二项：
  $\int \frac{\partial L}{\partial\dot{q}_i}\frac{d}{dt}(\delta q_i)\,dt = [\frac{\partial L}{\partial\dot{q}_i}\delta q_i]_{t_1}^{t_2} - \int \frac{d}{dt}(\frac{\partial L}{\partial\dot{q}_i})\delta q_i\,dt$

  \textbf{Step 5} — 边界项因固定端点消失 $\to$ 整理：
  $\delta S = \sum_i \int [\frac{\partial L}{\partial q_i} - \frac{d}{dt}(\frac{\partial L}{\partial\dot{q}_i})]\delta q_i\,dt$

  \textbf{Step 6} — $\delta S = 0$ (R2) + 变分学基本引理（$\delta q_i$ 任意）：
  $\frac{\partial L}{\partial q_i} - \frac{d}{dt}(\frac{\partial L}{\partial\dot{q}_i}) = 0$
  \hfill\texttt{[N]} $\delta q_i$ 任意且独立
\end{derivationbox}

\textbf{必要条件}：R2 ($\delta S=0$), $L \in C^2$, 固定端点, $\delta q_i$ 任意独立。

\textbf{充分条件}：给定 $C^2$ 拉格朗日量和固定端点，$\delta S = 0 \iff$ Euler-Lagrange 方程。\texttt{reversible: true}

\keypoint{Euler-Lagrange方程的核心：分部积分把对$\delta\dot{q}$的依赖转化为对$\delta q$的依赖——这本质上是把"速度的变分"转化为"位置的变分"，代价是引入一个时间导数$d/dt$作用在$\partial L/\partial\dot{q}$上。这个$d/dt(\partial L/\partial\dot{q})$就是"广义动量的变化率"——所以Euler-Lagrange方程说的是：广义力 = 广义动量的变化率。}

\bridge{$\delta S = 0 \xrightarrow{\text{分部积分} + \text{变分引理}} \frac{d}{dt}\frac{\partial L}{\partial\dot{q}} - \frac{\partial L}{\partial q} = 0$ \\ 分部积分把边界"扔掉"（固定端点），导数从变分$\delta q$转移到$\partial L/\partial\dot{q}$上}{自然界选择作用量最小的路径 $\to$ 这条路径必须满足一个二阶微分方程 $\to$ 给定初位置和初速度，未来完全确定 \\ 物理因果（决定论）从变分原理中自动涌现}

\begin{oombox}[数量级估计：经典作用量的大小]
  对于一个宏观物体，作用量 $S$ 的典型值远大于 $\hbar \approx 10^{-34}$ J·s：
  \begin{itemize}
    \item 棒球（$m=0.15$ kg, $v=40$ m/s, 飞行1秒）：$S \sim \frac{1}{2}mv^2 \cdot \Delta t \sim 120$ J·s $\sim 10^{36}\,\hbar$
    \item 电子在原子中（Bohr轨道）：$S \sim \hbar$（量子效应显著——不能用经典路径！）
    \item 宏观摆锤（$m=1$ kg, $l=1$ m, 周期2s）：$S \sim$ 几 J·s $\sim 10^{34}\,\hbar$
  \end{itemize}
  这就是为什么经典力学在宏观世界精确成立：$S/\hbar \gg 1$ 时，路径积分 $\int e^{iS/\hbar}$ 中只有 $\delta S=0$ 附近的路径相干加强，其他路径的贡献因相位快速振荡而抵消——这正是量子力学退化为经典力学的机制。
\end{oombox}

\section{Noether 定理}
% ════════════════════════════════════════════════════════

\subsection{物理直觉}

Emmy Noether 在 1918 年证明了一个被爱因斯坦称为"数学史上最深刻的定理之一"的结果。其物理含义极为简洁：

\begin{center}
\textbf{如果你在某个方向上看不出物理定律有任何区别，\\
那么在那一方向上就一定有一个守恒量。}
\end{center}

具体来说：
\begin{itemize}
  \item 时间平移看不出区别（今天和昨天物理定律一样）$\to$ \textbf{能量守恒}
  \item 空间平移看不出区别（这里和那里的物理定律一样）$\to$ \textbf{动量守恒}
  \item 旋转看不出区别（朝这个方向和那个方向的物理定律一样）$\to$ \textbf{角动量守恒}
  \item 量子相位平移看不出区别（波函数的整体相位不可观测）$\to$ \textbf{电荷守恒}
\end{itemize}

\textbf{数学结构}：Noether 定理是一个\textbf{数学恒等式}——只要你接受作用量原理和连续对称性，守恒流就自动出现。它不依赖于任何具体的力的形式、也不依赖于任何具体的物质组成。Hilbert 曾试图自己证明这个定理但没有成功；Noether 用群论和变分法结合的方式干净利落地完成——这需要同时精通抽象代数和分析力学，在当时的数学界几乎只有她能做到。

\begin{lawbox}[Noether 定理]
  作用量 $S$ 的每个连续对称性变换对应一个守恒的 Noether 荷 $Q$：
  \[
  Q \equiv \sum_i \frac{\partial L}{\partial \dot{q}_i}\,Q_i - F, \qquad \frac{dQ}{dt} = 0
  \]
  \textbf{前提}：R2, R1\quad \textbf{推导号}：\texttt{deriv.noether\_theorem}
\end{lawbox}

\begin{derivationbox}[推导：Noether 定理]
  \textbf{Step 1} — 对称性变换：$q_i \to q_i' = q_i + \varepsilon Q_i(q,\dot{q},t)$
  \hfill\texttt{[N]} $Q_i$ 可微，$\varepsilon$ 为 Lie 群参数

  \textbf{Step 2} — 对称性条件：$S[q'] = S[q] + \varepsilon[F(t_2)-F(t_1)]$
  $\implies L' = L + \varepsilon\,dF/dt$
  \hfill\texttt{[N]} 作用量必须不变（至多差边界项）

  \textbf{Step 3} — 展开 $L'$：$L' - L = \varepsilon\sum_i[\frac{\partial L}{\partial q_i}Q_i + \frac{\partial L}{\partial\dot{q}_i}\dot{Q}_i]$

  \textbf{Step 4} — 用 Euler-Lagrange 替换 $\partial L/\partial q_i$：
  $\frac{\partial L}{\partial q_i} = \frac{d}{dt}(\frac{\partial L}{\partial\dot{q}_i})$
  \hfill\texttt{[N]} 系统沿真实路径（on-shell）

  \textbf{Step 5} — 重组为全导数：
  $\sum_i[\frac{d}{dt}(\frac{\partial L}{\partial\dot{q}_i})Q_i + \frac{\partial L}{\partial\dot{q}_i}\dot{Q}_i] = \frac{d}{dt}(\sum_i \frac{\partial L}{\partial\dot{q}_i}Q_i)$

  \textbf{Step 6} — 与对称性条件联立：
  $\frac{d}{dt}(\sum_i \frac{\partial L}{\partial\dot{q}_i}Q_i - F) = 0$

  \textbf{Step 7} — 定义守恒荷：$Q \equiv \sum_i \frac{\partial L}{\partial\dot{q}_i}Q_i - F$，$dQ/dt = 0$
\end{derivationbox}

    \item \textbf{宏观机械能守恒}：精度 $\sim 10^{-5}$（受摩擦力等非保守力限制）
    \item \textbf{电磁学中的 Poynting 定理}：精度 $\sim 10^{-8}$（光子计数实验）
    \item \textbf{核反应中的质能守恒（$E=mc^2$ 结合）}：精度 $\sim 10^{-7}$（$\alpha$ 粒子能量测量）
    \item \textbf{宇宙学中的总能量}：广义相对论中全局能量守恒并不严格成立（因为时空中没有全局时间平移对称性）——这恰恰验证了 Noether 定理的逻辑：对称性不存在 $\Rightarrow$ 守恒律不存在
  \end{itemize}
  最后一个例子是 Noether 定理最重要的反面教材：膨胀宇宙没有时间平移对称性，所以宇宙的总能量没有严格守恒的定义。

\section{Newton 运动定律}

\subsection{第一定律（惯性定律）}

\subsubsection{物理直觉}

Newton 第一定律说"不受力的物体保持静止或匀速直线运动"。这听起来像废话——但它不是。在 Newton 之前，Aristotle 的物理学认为物体的"自然状态"是静止——推它才动，不推就停。第一定律的革命性在于：它声明\textbf{匀速运动}和静止一样"自然"，\textbf{改变速度才需要理由（力）}。

从 R2 的角度看，这更自然：自由粒子的 $L = \frac{1}{2}mv^2$（只有动能），Euler-Lagrange 方程给出 $d(m\mathbf{v})/dt = 0$——速度不变就是自由粒子的"自然选择"。

\keypoint{惯性定律不是实验总结出来的——它是对"力"的定义。什么样的参考系里自由粒子匀速运动？这个参考系就叫惯性系。所以第一定律同时定义了"惯性系"和"不受力"——它是一把双刃剑。}

\begin{oombox}[日常直觉与物理精确性]
  在桌面上推一本书，手离开后它停下来——Aristotle 说"自然状态是静止"。Newton 说：不对，它停下来是因为摩擦力——如果桌面完全光滑（真空、无摩擦），书会永远滑下去。\\
  \textbf{精度}：在地球表面，摩擦力 $f = \mu mg \approx 0.3 \times 1 \times 9.8 \approx 3$ N（对一本1kg的书）。在太空中（$f \ll 10^{-9}$ N），一个物体的速度可以保持在 $10^{-12}$ 精度以内。惯性定律在无外力时精确成立，偏差仅来自未被建模的微小外力。
\end{oombox}

\begin{lawbox}[Newton 第一定律]
  $F = 0 \implies d\mathbf{v}/dt = 0$（匀速直线运动）
  \quad\textbf{前提}：R2, R1\quad\textbf{推导号}：\texttt{deriv.newton\_first}
\end{lawbox}

\noindent\textbf{变量}：力 $F$ [M$\cdot$L$\cdot$T$^{-2}$], 速度 $\mathbf{v}$ [L$\cdot$T$^{-1}$], 质量 $m$ [M]（常数）。

\begin{derivationbox}[推导]
  \textbf{Step 1} — 自由粒子 Lagrangian：$L = \frac{1}{2}m|\mathbf{v}|^2$（$V$ = 常数 $\implies \nabla V = 0$）\\
  \textbf{Step 2} — Euler-Lagrange：$d(m\mathbf{v})/dt = -\nabla V = 0$
  \texttt{[N]} $m$ 常数\quad\texttt{[N]} 惯性系\\
  \textbf{Step 3} — $\mathbf{v} = $ 常数。\quad\texttt{[S]} 自由粒子 $L$ + Euler-Lagrange $\Rightarrow$ 第一定律。
\end{derivationbox}

\subsection{第二定律（运动方程）}

\begin{lawbox}[Newton 第二定律]
  $\mathbf{F} = d\mathbf{p}/dt$（一般形式）；$m$ 常数时 $\mathbf{F} = m\mathbf{a}$
  \quad\textbf{前提}：R2, Euler-Lagrange, R1
\end{lawbox}

\noindent\textbf{变量}：动量 $\mathbf{p} = m\mathbf{v}$ [M$\cdot$L$\cdot$T$^{-1}$], 加速度 $\mathbf{a} = d\mathbf{v}/dt$ [L$\cdot$T$^{-2}$],
力 $\mathbf{F} \equiv -\nabla V(\mathbf{r})$ [M$\cdot$L$\cdot$T$^{-2}$]。

\begin{derivationbox}[推导]
  \textbf{Step 1} — $L = \frac{1}{2}m|\mathbf{v}|^2 - V(\mathbf{r})$\quad\texttt{[N]} 保守力（$V$ 存在）\\
  \textbf{Step 2} — Euler-Lagrange：$d(m\mathbf{v})/dt = -\nabla V$\\
  \textbf{Step 3} — 定义 $\mathbf{F} \equiv -\nabla V \implies d\mathbf{p}/dt = \mathbf{F}$
  \quad\texttt{[S]} $L = T-V$ + Euler-Lagrange $\Rightarrow$ 第二定律。
\end{derivationbox}

\keypoint{$\mathbf{F} = m\mathbf{a}$ 不是 Newton 写的。他写的是 $\mathbf{F} = d\mathbf{p}/dt$——力的定义是"动量的变化率"。当质量不变时两者等价，但当质量可变（如火箭喷气、相对论效应），$\mathbf{F} = d\mathbf{p}/dt$ 才是正确形式。}

\bridge{$d(m\mathbf{v})/dt = -\nabla V \xrightarrow{\text{定义}} \mathbf{F} \equiv -\nabla V$ \\ 力不是原始概念——它是由势能的梯度\textbf{定义}出来的}{$\mathbf{F} = d\mathbf{p}/dt$ 不是实验规律——它是对"力"这个概念的\textbf{定义} \\ 真正有物理内容的是 $\mathbf{p}$ 和 $V$ 的形式，以及 $V$ 的可微性}

\begin{oombox}[第二定律的尺度范围]
  \begin{itemize}
    \item \textbf{原子尺度}（$r \sim 10^{-10}$ m）：Lorentz 力支配电子运动，$F \sim e^2/(4\pi\varepsilon_0 r^2) \approx 2.3 \times 10^{-8}$ N；量子效应显著（$S \sim \hbar$）
    \item \textbf{人体尺度}（$m \sim 70$ kg，$a \sim 1$ m/s$^2$）：$F \sim 70$ N；经典力学精确成立
    \item \textbf{太阳系尺度}（$M_\odot \sim 2\times10^{30}$ kg，$r \sim 1$ AU $\sim 1.5\times10^{11}$ m）：$F = GM_\odot M_\oplus/r^2 \approx 3.5\times10^{22}$ N；牛顿引力精确度 $\sim 10^{-8}$（仅水星近日点进动需 GR 修正，$\sim 43''$ / 世纪）
    \item \textbf{速度极限}：$v \ll c$ 时第二定律（$F=ma$）成立；$v \to c$ 时需替换为 $F = d(\gamma mv)/dt$
  \end{itemize}
\end{oombox}

\subsection{第三定律（作用-反作用）}

\begin{lawbox}[Newton 第三定律]
  $\mathbf{F}_{12} = -\mathbf{F}_{21}$（弱形式：大小相等方向相反；强形式 collinear 在 EM 中破缺）
  \quad\textbf{前提}：Noether 定理, R1
\end{lawbox}

\begin{derivationbox}[推导]
  \textbf{Step 1} — 孤立系统动量守恒（Noether）：$d(\mathbf{p}_1 + \mathbf{p}_2)/dt = 0$
  \texttt{[N]} 孤立系统（无外力）\\
  \textbf{Step 2} — Newton 第二定律：$d\mathbf{p}_1/dt = \mathbf{F}_{12}$，$d\mathbf{p}_2/dt = \mathbf{F}_{21}$\\
  \textbf{Step 3} — $\mathbf{F}_{12} + \mathbf{F}_{21} = 0 \implies \mathbf{F}_{12} = -\mathbf{F}_{21}$
  \quad\texttt{[S]} 动量守恒 + 第二定律 $\Rightarrow$ 第三定律弱形式。
\end{derivationbox}

\keypoint{电磁学中，两个运动的点电荷之间的力并不严格满足 Newton 第三定律（强形式要求力沿连线），因为电磁场本身携带动量。系统的总动量（粒子 + 场）仍然守恒——这是 Noether 定理的又一个精妙之处：守恒的是总动量，不是粒子的机械动量。}

\precisionbox[Newton 三定律：精度等级]
  \textbf{第一定律}：\precexact（惯性系的定义性特征），但在 GR 中惯性系仅是局域概念\\
  \textbf{第二定律}：\preceffective——$L = T-V$ 是有效描述，非相对论性（$v \ll c$），非量子性（$S \gg \hbar$）\\
  \textbf{第三定律——弱形式}（$\mathbf{F}_{12} = -\mathbf{F}_{21}$）：\preceffective——在电磁动力学中，场携带的动量使瞬时作用的第三定律破缺\\
  \textbf{第三定律——强形式}（力沿连线）：\precisionapprox——电磁力不沿连线（Biot-Savart 定律中运动电荷间的力有横向分量）
\endprecisionbox

\subsection{万有引力定律}

\begin{lawbox}[Newton 万有引力定律]
  $\mathbf{F}_{12} = -G\,\dfrac{m_1 m_2}{r^2}\,\hat{\mathbf{r}}$
  \quad\textbf{前提}：Einstein 场方程（弱场极限）, R1
\end{lawbox}

\noindent\textbf{变量}：引力常数 $G = 6.67430(15)\times10^{-11}$ m$^3\cdot$kg$^{-1}\cdot$s$^{-2}$ [M$^{-1}\cdot$L$^3\cdot$T$^{-2}$],
引力势 $\phi = -GM/r$ [L$^2\cdot$T$^{-2}$], 质量密度 $\rho$ [M$\cdot$L$^{-3}$]。

\begin{derivationbox}[推导：从 Einstein 场方程到 Newton 引力]
  \textbf{Step 1} — 弱场近似：$g_{\mu\nu} = \eta_{\mu\nu} + h_{\mu\nu}$，$|h_{\mu\nu}| \ll 1$
  \texttt{[N]} weak\_field\_limit\\
  \textbf{Step 2} — Einstein 场方程 (0,0) 分量弱场展开：$R_{00} \approx -\frac{1}{2}\nabla^2 h_{00}$\\
  \textbf{Step 3} — 从测地线方程得 $h_{00} = 2\phi/c^2$\\
  \textbf{Step 4} — 化为 Poisson 方程：$\nabla^2\phi = 4\pi G\rho$\\
  \textbf{Step 5} — 球对称解：$\phi = -GM/r$，$\mathbf{F} = -m\nabla\phi = -GmM/r^2\,\hat{\mathbf{r}}$
  \quad\texttt{[S]} Einstein 场方程 + 弱场低速极限 $\Rightarrow$ Newton 引力。
  \texttt{reversible: false}（Newton 理论是 GR 的低能有效理论）。
\end{derivationbox}

\keypoint{引力常数 $G$ 是所有物理常数中测量精度最低的——不确定性 $2.2 \times 10^{-5}$（CODATA 2022）。相比之下，精细结构常数 $\alpha$ 的精度是 $1.5 \times 10^{-10}$。引力太弱了——两个质子间的引力是电磁力的 $Gm_p^2/(e^2/4\pi\varepsilon_0) \approx 8 \times 10^{-37}$——这使得精确测量 $G$ 极其困难。}

\bridge{Einstein 场方程 $G_{\mu\nu} = (8\pi G/c^4)T_{\mu\nu}$ \\ $\xrightarrow{\text{弱场} + \text{低速}}$ $\nabla^2\phi = 4\pi G\rho \xrightarrow{\text{球对称}}$ $\mathbf{F} = -GmM/r^2\,\hat{\mathbf{r}}$}{GR $\to$ Newton 引力的过程不是"近似"——它是 GR 在特定极限下的\textbf{精确约化} \\ 每步都是数学恒等式在特定条件下的退化，而非数值拟合}

\begin{oombox}[引力的尺度——从量子到宇宙]
  \begin{itemize}
    \item \textbf{两个质子间}（$r=10^{-15}$ m）：$F_{\text{grav}} \approx 1.9 \times 10^{-34}$ N vs.\ $F_{\text{EM}} \approx 230$ N ——引力比电磁力弱 $10^{36}$ 倍
    \item \textbf{两个1kg铅球间}（$r=0.1$ m）：$F_{\text{grav}} = G\cdot 1\cdot 1 / 0.01 \approx 6.7 \times 10^{-9}$ N ——约等于一粒沙子的重量（Cavendish 1798 年用扭秤测到了这个力！）
    \item \textbf{地球表面}（$M_\oplus=5.97\times10^{24}$ kg, $R_\oplus=6.37\times10^6$ m）：$g = GM_\oplus/R_\oplus^2 \approx 9.81$ m/s$^2$ ——每个人每天感受到的引力
    \item \textbf{太阳表面}：$g_\odot = GM_\odot/R_\odot^2 \approx 274$ m/s$^2$ ——约28倍地球重力
    \item \textbf{中子星表面}（$M=1.4M_\odot$, $R=10$ km）：$g \approx 1.9 \times 10^{12}$ m/s$^2$ ——GR 修正必须考虑（$GM/Rc^2 \approx 0.2$）
    \item \textbf{事件视界}（黑洞 $M=M_\odot$, $r_s=2GM/c^2 \approx 3$ km）：Newton 引力完全失效，GR 是唯一描述
  \end{itemize}
\end{oombox}

\precisionbox[精度等级]
  Newton 万有引力：\precisionapprox——它是 GR 的弱场低速极限。\\
  适用范围：$|\phi|/c^2 \ll 1$（在地球表面 $|\phi|/c^2 \approx 7\times 10^{-10}$），$v \ll c$。\\
  超出范围时失效：水星近日点进动（第一个 GR 验证）、GPS 时间修正（每天 $\sim 38\,\mu$s）、黑洞、引力波。
\endprecisionbox

\begin{lawbox}[Kepler 第三定律 — corollary]
  $T^2 \propto r^3$（轨道周期的平方正比于半长轴的立方）
  \quad\textbf{前提}：Newton 第二 + 万有引力, R1
\end{lawbox}

\noindent\textbf{变量}：轨道周期 $T$ [T], 圆轨道半径 $r$ [L]。

\begin{derivationbox}[推导]
  \textbf{Step 1} — 圆轨道向心加速度：$a = v^2/r$；Newton 第二：$F = mv^2/r$
  \texttt{[N]} 圆轨道近似（$e=0$）\\
  \textbf{Step 2} — 万有引力提供向心力：$mv^2/r = GMm/r^2 \implies v^2 = GM/r$\\
  \textbf{Step 3} — 轨道周期：$T = 2\pi r/v \implies T^2 = (4\pi^2/GM)\,r^3$
  \quad\texttt{[S]} Newton 力学 + 万有引力 $\Rightarrow$ Kepler 第三定律。
\end{derivationbox}

\keypoint{Kepler 用了十年分析 Tycho Brahe 的观测数据才发现 $T^2 \propto r^3$。而从 Newton 力学出发，推导只需要三行——这就是"从第一性原理出发"的力量。但注意：Newton 的推导假设了轨道是圆形的；Kepler 的原始版本适用于椭圆轨道（$T^2 \propto a^3$，其中 $a$ 是半长轴），这需要额外的角动量守恒论证才能从 Newton 力学推出。}

\section*{反事实分析}

\begin{limitbox}[若无最小作用量原理]
  Euler-Lagrange 方程消失 $\to$ 所有动力学方程失去统一机制。Noether 定理失效 $\to$ 能量、动量、角动量守恒消失。量子力学路径积分 $Z = \int \mathcal{D}\phi\, e^{iS/\hbar}$ 失去物理基础。
\end{limitbox}

\begin{limitbox}[若 G 的值不同]
  $G$ 大 1 倍：恒星寿命缩短至 1/4。$G=0$：无引力聚集 $\to$ 无恒星、行星、生命。$G$ 负值：引力变斥力 $\to$ 宇宙均匀膨胀，无结构。
\end{limitbox}

\section*{习题}
\addcontentsline{toc}{section}{习题}

\begin{enumerate}
  \item \textbf{[简单]} 从 $L = \frac{1}{2}m(\dot{x}^2 + \dot{y}^2 + \dot{z}^2) - V(x,y,z)$ 写出三个 Euler-Lagrange 方程，证明等价于 $m\ddot{\mathbf{r}} = -\nabla V$。
  \item \textbf{[简单]} 一维谐振子 $L = \frac{1}{2}m\dot{x}^2 - \frac{1}{2}kx^2$。推导 $\ddot{x} + \omega^2 x = 0$（$\omega = \sqrt{k/m}$），给出通解。
  \item \textbf{[中等]} 证明：若 $\partial L/\partial t = 0$，则 $H = \sum p_i \dot{q}_i - L$ 守恒。对 $L = \frac{1}{2}m\dot{x}^2 - V(x)$，何时 $H$ 等于总能量？
  \item \textbf{[中等]} 用 Noether 定理：若 $V$ 仅依赖粒子间相对距离，证明总动量守恒。推广到 $N$ 个粒子？
  \item \textbf{[中等]} 从 $T^2 = 4\pi^2 a^3/[G(M+m)]$，用太阳质量 $M_\odot = 1.989\times10^{30}$ kg 和地球轨道参数验证 $G = 6.674\times10^{-11}$。
  \item \textbf{[较难]} 证明：在 $D$ 维空间中，$F(r) \propto 1/r^{D-1}$ 是 Gauss 定律的推论。若 $D=2$，引力和 Kepler 第三定律分别变成什么？
  \item \textbf{[挑战]} GR 中 $L = -mc^2\sqrt{1-v^2/c^2}$。展开 $\gamma mc^2 \approx mc^2 + \frac{1}{2}mv^2 + \frac{3}{8}mv^4/c^2 + \cdots$。水星近日点进动（$43''$/世纪）来自 $v^4/c^4$ 阶修正。取 $v \approx 48$ km/s，估算此修正相对于 Newton 项的比值。
\end{enumerate}
