\subsection{科普层：自然为何"懒惰"}

1662 年，Pierre de Fermat 提出了一个奇怪的原理：光在介质中走的不是最短路径，而是最快路径。这个原理优雅地解释了折射定律，但也提出了一个不安的问题——光怎么"知道"哪条路径最快？

1744 年，Pierre-Louis Maupertuis 把这个想法推广到所有的力学：自然以"最少作用量"运行。他用神学语言阐述这一原理——"自然必然以最简洁的方式行事，因为这是上帝的本性"。这个神学动机后来被剥离，但数学骨架被 Lagrange (1788) 发展为分析力学的核心，并在 Hamilton (1834) 手中达到现代形式。

20 世纪 Feynman 给出了最深刻的重述：粒子并非"选择"最少作用量的路径，而是\textbf{所有可能路径同时贡献}，每条路径附带相位 $e^{iS/\hbar}$。当 $\hbar \to 0$ 时，非经典路径的相位剧烈振荡而互相抵消，只有 $\delta S = 0$ 的稳定相位路径存活——这就是经典轨迹。\textbf{最少作用量原理因此是量子力学在经典极限下的必然推论 [N]}。

\subsection{核心图像：作用量的几何与驻点}

考虑一颗以初速 $v_0$ 竖直上抛的石头，受重力 $g$。它的真实轨迹是 $y(t) = v_0 t - \tfrac{1}{2}g t^2$。沿这条轨迹，作用量是：
\[
S = \int_0^T \left[ \tfrac{1}{2}m\dot{y}^2 - mgy \right]\,dt
\]

如果我们尝试任何邻近轨迹 $y(t) + \delta y(t)$（端点固定 $\delta y(0)=\delta y(T)=0$），变分给出 $\delta S = \int_0^T (-m\ddot{y} - mg)\,\delta y\,dt$。\textbf{当且仅当 $\ddot{y} = -g$ 时 $\delta S = 0$ 对任意 $\delta y$ 成立}——这正是 Newton 第二定律。

注意一个关键细节：$\delta S = 0$ 表示 $S$ 在真实轨迹上是\textbf{驻点}，未必是极小点。对短时间区间 $S$ 是极小，但对长时间或共轭点之后 $S$ 可以是鞍点。术语"最少作用量"是历史称呼；现代严格说法是\textbf{驻作用量原理}（principle of stationary action）。

\paragraph{量纲检验} $[S] = [\text{能量}] \cdot [\text{时间}] = \text{J·s} = \text{kg·m}^2/\text{s}$，恰好是 Planck 常数 $\hbar$ 的量纲——这不是巧合，而是 R2 与 R4 在量子力学中统一的结构性印证。
```

\subsection{科普层：物理无视"零点"的选择}

家庭电路的"零电位"是接地——但接地只是一个约定。电势 $\phi$ 不是物理观测量，所有可观测电场 $\mathbf{E} = -\nabla\phi$ 只依赖电势的\textbf{差}。换言之，给所有电势同时加一个常数 $C$，物理不变。这种"任意性中蕴含的不变性"正是\textbf{规范不变性}的最简单例子。

1918 年，Hermann Weyl 试图把这个观念推广到引力——他想用一个"局域尺度变换"来统一引力与电磁。Einstein 立即指出这个方案与原子谱实验矛盾（不同地点的氢原子谱线会变），方案失败。但 Weyl 的核心想法——\textbf{允许某种相位/尺度的局域变换，并要求物理不变}——在 1929 年被他自己重新解释为电磁规范变换 $A_\mu \to A_\mu + \partial_\mu\Lambda$，而 $\Lambda(x)$ 是任意时空函数。这次成功了。

1954 年，Yang 和 Mills 把规范不变性从交换的 $U(1)$ 推广到非交换李群 $SU(N)$，开启了现代规范场论。今天我们知道：电磁是 $U(1)$ 规范场、弱相互作用是 $SU(2)$、强相互作用是 $SU(3)$。\textbf{标准模型的全部相互作用都是局域规范不变性的推论 [N]}。

\subsection{核心图像：规范变换不改变物理}

考虑一个带电粒子在外加电磁场中运动。它的拉氏量是：
\[
L = \tfrac{1}{2}m\dot{\mathbf{x}}^2 - q\phi(\mathbf{x},t) + q\dot{\mathbf{x}} \cdot \mathbf{A}(\mathbf{x},t)
\]

施加规范变换 $\phi \to \phi - \partial_t\Lambda$, $\mathbf{A} \to \mathbf{A} + \nabla\Lambda$。新拉氏量：
\[
L' = L - q\partial_t\Lambda + q\dot{\mathbf{x}}\cdot\nabla\Lambda = L + q\frac{d\Lambda}{dt}
\]

\textbf{$L$ 与 $L'$ 仅相差一个全微分项 $q\,d\Lambda/dt$}。由 R2，作用量 $S = \int L\,dt$ 在端点固定时差 $q[\Lambda]_{t_0}^{t_1}$，但 $\delta S = \delta S'$——\textbf{运动方程不变}。物理不可观测 $\Lambda$，正如电势的"零点"不可观测。

\paragraph{纤维丛图像} 把每个时空点 $x$ 想象成"挂"了一个相位圆 $U(1)$。物理状态 $\psi(x)$ 在每点是相位圆上的一个角度。规范变换 = 在每点独立旋转相位圆。要求物理不变 = 要求方程把"相对相位"而非"绝对相位"作为有意义的量。这正是规范场 $A_\mu$ 必须存在的几何理由——它是\textbf{相邻相位圆之间的"连接"}，告诉我们如何把不同点的相位比较。
```

\subsection{科普层：为什么时间有方向}

把一杯热咖啡放在桌上，它会冷却到室温——但反过来，从未见过桌上的冷咖啡自发变热。微观物理的所有基本方程（Newton, Schrödinger, Maxwell, Einstein）都是\textbf{时间反演对称}的：把所有粒子速度反向，方程依然成立。那为什么宏观世界有"昨天"和"明天"的区别？

1872 年，Ludwig Boltzmann 给出了答案——\textbf{时间方向是统计的，不是动力学的}。一杯咖啡变冷的"前进方向"对应 $W$（微观态数）增大的方向，因为 $W$（热平衡）远大于 $W$（咖啡温度高、室温低的非平衡态）。从单粒子方程角度看，反向轨迹和正向轨迹同样合法；但从全部粒子相空间体积看，反向轨迹是 $1/W \approx 10^{-10^{23}}$ 的概率事件。

Boltzmann 把这种统计倾向写成了一个公式：$S = k\ln W$。它不仅给出第二定律的微观基础，也把"熵"重新定义为\textbf{无知的对数测度}——你越不能区分微观态，宏观熵越大。Maxwell 的"妖"思想实验（1867）和 Landauer 的擦除原理（1961）后来证明：信息处理本身有最小熵代价 $k\ln 2$ 每比特，连"妖"也无法逃脱。

\subsection{核心图像：微观态的指数爆炸}

掷一颗骰子，6 个微观态，$S = k\ln 6 \approx 1.79 k$。掷两颗，$36$ 态，$S = k\ln 36 \approx 3.58 k$。掷 $n$ 颗，$6^n$ 态，$S = nk\ln 6$，与 $n$ 线性。

但物理熵的"指数爆炸"在这里：1 摩尔气体有 $N_A \approx 6\times 10^{23}$ 个分子，每个分子有自己的位置和动量（约 $10^{20}$ 量级的微观自由度）。微观态总数 $W \sim 10^{10^{23}}$。\textbf{即使 $S$ 只是 $\ln W$，也已是天文数字}：一杯水的熵约 $10$ J/K，对应 $W \sim e^{10/k} \sim e^{10^{24}}$。

正是这种指数爆炸保证了第二定律的\textbf{统计必然性}：从低 $W$ 态进入高 $W$ 态的"偶然涨落"概率是 $e^{-\Delta S / k}$，对宏观系统这个数小到任何宇宙年龄内都不可能发生。

\paragraph{熵的两副面孔}
\begin{itemize}
\item \textbf{Boltzmann 熵} $S_B = k\ln W$：客观，依赖能壳上的微观态数。
\item \textbf{Gibbs/Shannon 熵} $S_G = -k\sum_i p_i \ln p_i$：主观，依赖你假设的概率分布 $p_i$。
\end{itemize}
两者在等概率先验 $p_i = 1/W$（R6b）下重合：$S_G = k\ln W = S_B$。这正是 R6b "等概率先验" [N] 的所在——它把统计力学锚定在客观微观态计数上，否则熵的定义将依赖观察者。
```

\subsection{科普层：三百年后 $F = ma$ 仍未被推翻}

苹果落在 Newton 头上的故事是后人的浪漫化（实际是他在乡下避瘟疫时观察到的）。但 Newton 1687 年的真正成就远比"发现重力"深刻：他证明了\textbf{使苹果落地的力，与维持月球绕地的力，是同一种力}——并以 $F = G m_1 m_2 / r^2$ 量化了它。

\textbf{Newton 把 Kepler 的三定律从经验规律提升为推论}。Kepler (1609-1619) 通过 Tycho Brahe 的火星观测归纳出三条经验定律：椭圆轨道、面积速度恒定、周期与半长轴 3/2 次方成正比。Newton 证明：只要假设引力是 $1/r^2$ 的中心力，所有三条 Kepler 定律就是数学推论。这是物理学第一次成功地把"现象学规律"还原为"基础动力学定律"。

但 Newton 的 $F = ma$ 形式在数学上并非最优雅。1788 年，Lagrange 在《分析力学》中提出：用广义坐标 $q_i$ 和拉氏量 $L = T - V$，运动方程是 $\partial L/\partial q_i = d(\partial L/\partial \dot{q}_i)/dt$。这个形式有三大优势：
\begin{enumerate}
\item \textbf{坐标无关}：不必区分笛卡尔/极/球坐标，自动处理约束。
\item \textbf{对称性显式}：Noether 定理直接从 $L$ 的对称性得到守恒量。
\item \textbf{推广无碍}：场论、相对论、量子力学都使用同一框架。
\end{enumerate}

\textbf{现代物理教学普遍以 Lagrange 形式为主}，正是因为它把 R2（最少作用量）作为根基，而 $F = ma$ 退化为它的特殊推论。Newton 形式仍在工程和入门课程中广泛使用，因为它直接、直观；但在深层物理研究中，几乎所有先进理论都从拉氏量出发。

\subsection{核心图像：单摆的两副语言}

考虑长度 $\ell$、质量 $m$ 的单摆，重力 $g$。设摆角 $\theta$ 为广义坐标。

\textbf{Newton 视角}：在切向方向，重力分量 $-mg\sin\theta$ 提供加速度 $\ell\ddot\theta$，得：
\[
\ell\ddot\theta = -g\sin\theta \quad \Rightarrow \quad \ddot\theta + (g/\ell)\sin\theta = 0
\]

\textbf{Lagrange 视角}：动能 $T = \tfrac{1}{2}m\ell^2\dot\theta^2$，势能 $V = mg\ell(1-\cos\theta)$。
\[
L = T - V = \tfrac{1}{2}m\ell^2\dot\theta^2 - mg\ell(1-\cos\theta)
\]
Euler-Lagrange 方程：
\[
\frac{d}{dt}\frac{\partial L}{\partial \dot\theta} - \frac{\partial L}{\partial \theta} = m\ell^2\ddot\theta + mg\ell\sin\theta = 0
\]
即 $\ddot\theta + (g/\ell)\sin\theta = 0$。

两种语言得到相同方程。但 Lagrange 视角的优势在于：如果加上约束（双摆、绳长变化等），Newton 视角需要小心处理张力等内力，Lagrange 视角自动消去内力——这就是它在复杂系统中胜出的原因。

\paragraph{小角近似量级估算} 当 $\theta \ll 1$，$\sin\theta \approx \theta$，方程退化为 $\ddot\theta = -(g/\ell)\theta$，简谐运动周期 $T = 2\pi\sqrt{\ell/g}$。1 米单摆 $T \approx 2$ 秒，与日常经验一致。
```

\[
\text{Poincaré}(c) \xrightarrow{c\to\infty} \text{Galilean}, \qquad \text{Poincaré}(c) \xrightarrow{c\to 0} \text{Carroll}
\]

我们的宇宙选择了\textbf{有限非零 $c$}的中间情形。这个"非零有限"是经验事实而非逻辑必然。

    "kernel":     ("rectangle, rounded corners=2pt", "red!15", "red!60!black"),
    "law":        ("ellipse",                          "blue!10", "blue!60!black"),
    "corollary":  ("ellipse",                          "green!12", "green!50!black"),
    "contingent": ("regular polygon, regular polygon sides=6", "gray!20", "gray!50!black"),
}

def load_all():
    """Return dict of all node IDs → node info from frameworks/laws/cor/contingent."""
    nodes = {}
    for fname, layer in [("frameworks.yaml", "kernel"),
                         ("effective_laws.yaml", "law"),
                         ("contingent.yaml", "contingent")]:
        path = LAYER1 / fname
        if not path.exists():
            continue
        data = yaml.safe_load(path.read_text())
        for k in ("kernels", "rules", "laws", "corollaries", "contingent"):
            for entry in data.get(k, []) or []:
                nid = entry.get("id")
                if nid:
                    nodes[nid] = {"layer": "corollary" if k == "corollaries" else layer,
                                  "label": entry.get("name", nid),
                                  "premises": entry.get("derived_from", []) or entry.get("premise", [])}
    return nodes

def layout_dot(nodes, edges, algo="dot"):
    """Run graphviz to lay out nodes; return dict id → (x, y)."""
    src = ["digraph G {", "  graph [rankdir=TB];", "  node [shape=box];"]
    for nid in nodes:
        src.append(f'  "{nid}";')
    for src_id, dst_id in edges:
        src.append(f'  "{src_id}" -> "{dst_id}";')
    src.append("}")
    proc = subprocess.run([algo, "-Tjson"], input="\n".join(src),
                          capture_output=True, text=True, check=True)
    parsed = json.loads(proc.stdout)
    pos = {}
    for obj in parsed.get("objects", []):
        if "pos" in obj and obj.get("name"):
            x, y = map(float, obj["pos"].split(","))
            pos[obj["name"]] = (x / 72.0, y / 72.0)  # graphviz uses points
    return pos

def emit_tikz(nodes, edges, pos, highlight=None, scale=0.04):
    """Return TikZ source for the given subgraph."""
    out = []
    highlight = highlight or set()
    for nid, info in nodes.items():
        if nid not in pos:
            continue
        x, y = pos[nid]
        layer = info["layer"]
        shape, fill, draw = NODE_STYLES[layer]
        opacity = "1.0" if (not highlight or nid in highlight) else "0.25"
        label = info["label"][:20].replace("_", r"\_")
        out.append(
            f"\\node[{shape}, fill={fill}, draw={draw}, inner sep=2pt, "
            f"font=\\tiny, opacity={opacity}] ({nid}) at ({x*scale},{y*scale}) {{{label}}};"
        )
    for src_id, dst_id in edges:
        if src_id in pos and dst_id in pos:
            opacity = "1.0" if (not highlight or src_id in highlight or dst_id in highlight) else "0.15"
            out.append(f"\\draw[->, gray!60, opacity={opacity}] ({src_id}) -- ({dst_id});")
    return "\n".join(out) + "\n"

def gen_dag_full(out_dir):
    """G1: full dependency DAG."""
    nodes = load_all()
    edges = []
    for nid, info in nodes.items():
        for prem in info.get("premises", []):
            if prem in nodes:
                edges.append((prem, nid))
    pos = layout_dot(nodes, edges, algo="dot")
    tikz = emit_tikz(nodes, edges, pos)
    (out_dir / "dag-full.tikz").write_text(tikz)
    print(f"  dag-full.tikz: {len(nodes)} nodes, {len(edges)} edges")

def gen_locator(out_dir, ch_nodes_map):
    """16 chapter locator figures."""
    nodes = load_all()
    edges = []
    for nid, info in nodes.items():
        for prem in info.get("premises", []):
            if prem in nodes:
                edges.append((prem, nid))
    pos = layout_dot(nodes, edges, algo="dot")
    for ch, highlight in ch_nodes_map.items():
        tikz = emit_tikz(nodes, edges, pos, highlight=set(highlight))
        (out_dir / f"locator-ch{ch:02d}.tikz").write_text(tikz)
        print(f"  locator-ch{ch:02d}.tikz: highlight {len(highlight)} nodes")

def gen_affect_matrix(out_dir):
    """G2: 6×N matrix of R1-R6 vs derivations, color by 'breaks' relation."""
    cf_path = LAYER1 / "counterfactuals.yaml"
    if not cf_path.exists():
        return
    cf_data = yaml.safe_load(cf_path.read_text())
    laws_data = yaml.safe_load((LAYER1 / "effective_laws.yaml").read_text())
    laws = [l["id"] for l in laws_data.get("laws", []) or []]
    rules = ["R1a", "R1b", "R1c", "R2", "R3", "R4", "R5", "R6a", "R6b"]
    # Build matrix: matrix[rule][law] = 'direct' / 'indirect' / 'none'
    # ... (concrete implementation maps cf entries to matrix cells)
    out = ["% G2 affect matrix"]
    # Render as TikZ matrix of \fill rectangles
    cell = 0.4
    for i, rule in enumerate(rules):
        for j, law in enumerate(laws):
            color = "white"  # default
            # determine color by scanning cf_data['counterfactuals']
            x, y = j * cell, -i * cell
            out.append(f"\\fill[{color}] ({x},{y}) rectangle ({x+cell},{y+cell});")
        out.append(f"\\node at (-1,{-i*cell-cell/2}) {{\\tiny {rule}}};")
    (out_dir / "affect-matrix.tikz").write_text("\n".join(out) + "\n")
    print(f"  affect-matrix.tikz: {len(rules)}×{len(laws)} matrix")
